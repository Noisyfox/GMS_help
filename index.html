欢迎浏览 GameMaker:Studio 用户手册! 这个文档分为三个部分，主要讲解 GameMaker:Studio 界面和基础使用，更高级的用法和功能需要使用 GML (the GameMaker Language). 新用户必须从 Using GameMaker:Studio 部分开始，这部分包括所有快速入门知识, 曾经使用过 GameMaker 的用户可以跳过这部分，请查看 Advanced Use 部分关于新版本更新内容。不论您的技术水平如何，我们都希望这本说明能够帮助您有效的使用 GameMaker:Studio 并且实现您的梦！

Welcome to the GameMaker:Studio user manual! This document is divided into three parts with the aim of getting you introduced to the interface and basic workings of GameMaker:Studio before going on to more advanced usage and the functions available through GML (the GameMaker Language). New users should definitely start at the beginning with the Using GameMaker:Studio section as it has all the information you need to get started as quickly as possible, however users of previous GameMaker versions can probably skip to the Advanced Use section to see what's new in this version. Whatever your skill level, we hope that this manual helps you to realise the true power within GameMaker:Studio and make your gaming dreams become reality![收起英文说明]

NOTE : 由于 GameMaker:Studio 模块化性质，一些技巧是不包括在手册中，特别是特殊设定和连接不同的设备方面。对于最新的信息和故障排除提示和技巧，请参阅 YoYo Games Wiki.
