Instance Functions 实例函数

本章节涉及专门处理实例的函数。

实例是对象资源的拷贝，放置在房间中进行互动，并做一些事情来让你的游戏活起来。如下是处理实例的函数。

  1. instance_change
  2. instance_copy
  3. instance_count
  4. instance_create
  5. instance_destroy
  6. instance_exists
  7. instance_find
  8. instance_furthest
  9. instance_id
  10. instance_nearest
  11. instance_number
  12. instance_place
  13. instance_position
