instance_deactivate_all

Deactivates all instances in the room.

Syntax :

instance_deactivate_all(notme);


ArgumentDescription
notme Whether to keep the calling instance activated (true) or not (false).


Returns: N/A


Description

This function will deactivate all instances that are in the room at the moment that the code is run. This may include the instance running the code if the "notme" flag is set to false, but normally you would want that instance to be active, in which case the "notme" flag should be set to true.


Example :

instance_deactivate_all(true);
instance_activate_region(view_xview[0] - 64, view_yview - 64, view_wview[0] + 128, view_hview + 128, false, false);

The above code deactivates all instances except the one that is running the code and then actiavtes a region within the room.
