Generating Events 生成事件

如下列出了可用的处理对象的事件的函数。

正如你所知道的，GameMaker完全是事件驱动的，而且所有行动的结果有一个特定的顺序，或在游戏中发生的特定事件而触发某些事件发生。这里有许多不同的事件，每一个与一个特定的任务或定时，其中一些甚至分割成单独的“子”的事件。正常情况下，这些事件的运行是通过GameMaker:Studio检测到一个游戏中的触发器，或每场游戏的标记（如步骤、绘制事件的情况下），但你也可以在当前的房间内的一段代码中申请一个事件到一个实例。


下面的函数存在为处理生成事件：

  1. event_inherited
  2. event_perform
  3. event_perform_object
  4. event_user

也有一些只读变量，​​可以被使用在以上所列的函数中，作为调试是特别有用：

  1. event_action
  2. event_number
  3. event_object
  4. event_type
