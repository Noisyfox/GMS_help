event_action

The index of the action that is currently being executed.

Syntax :

event_action;


Returns : Real


Description

This read-only variable returns the index of the action currently being executed, starting at 0. So, if you have an event with three action blocks in it, then check this variable on the third, it will return 2.


Example :

num = event_action;

The above code stores the current action number in the variable "num".
