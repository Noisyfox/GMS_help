object_set_mask

Sets the mask index of the given object.

Syntax :

object_set_mask(index, spr);


ArgumentDescription
index The index of the object to change.
spr The new sprite to use as the object's mask.


Returns : N/A


Description

With this function you can set the mask_index of a specific object (for more information on masks see Defining Objects). This means that all instances of this object that are created after the mask has been changed will be created with this new mask, while instances that are already in the room may not be affected. Please note that this is not an instance function! You can set the mask index of individual instances and so have ten instances with ten different masks, but if you change the mask index of the object, all instances that are created of that object will start with that new mask. You can set this to -1 to remove a mask sprite and so default to the mask defined for the sprite of the object (or no masks if no sprite has been chosen).


Example :

if global.level=10 object_set_mask(obj_Platform, spr_mask_10);

The above code checks the value of global variable and then changes the mask index of the object "obj_Platform" if it is equal to ten.
