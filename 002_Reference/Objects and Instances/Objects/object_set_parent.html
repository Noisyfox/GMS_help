object_set_parent

Sets the parent of the given object.

Syntax :

object_set_parent(index, obj);


ArgumentDescription
index The index of the object to change.
obj The new parent.


Returns : N/A


Description

With this function you can set the parent of a specific object (a parent is another object that can be used to "share" properties with its children. See More About Objects : Parents for more information). This means that all instances of this object that are created after the parent has been changed will be created with this new parent, while instances that are already in the room may not be affected. Please note that this is not an instance function! You cannot set the parent of individual instances, only of the object and so changing this while there are instances in the room is not recommended and can lead to unpredictable results. You may also set this to -1 to remove parents from the object.


Example :

if !instance_exists(obj_Enemy) object_set_parent(obj_Enemy, -1);

The code above will remove all parents from the object "obj_Enemy" if no instances of the object "obj_Enemy" exist in the current room.
