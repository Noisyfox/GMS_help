os_type

This variable holds a different constant depending on the operating system the game is currently being run with.

Syntax :

os_type;


Returns : Constant


Description

This variable holds one of various constant GameMaker:Studio has to tell you which operating system the game is being run on. The following constants can be returned :

ConstantDescription
os_windowsWindows OS
os_win8nativeWindows 8
os_win32Windows 32bit OS
os_winphoneWindows 8 Phone
os_linuxLinux
os_macosxMac OS X
os_iosiOS (iPhone, iPad, iPod Touch)
os_androidAndroid
os_unknownunknown OS

Example :

switch (os_type)
   {
   case os_windows: global.Config = 0; break;
   case os_android: global.Config = 1; break;
   case os_linux: global.Config = 2; break;
   case os_macosx: global.Config = 3; break;
   case os_ios: global.Config = 4; break;
   case os_winphone: global.Config = 5; break;
   }

The above code checks to see if the game is running in a browser or not and sets a global variable to a value depending on the result of the check.
