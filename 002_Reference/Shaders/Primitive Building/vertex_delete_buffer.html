vertex_delete_buffer

Delete a vertex buffer from memory.

Syntax :

vertex_delete_buffer(buffer);


ArgumentDescription
buffer The buffer to be removed.


Returns: N/A


Description

This function can be used to remove a previously created vertex buffer (see vertex_create_buffer) from system memory. This will also remove all vertex data for any custom primitives that it contained.


Example :

v_buff = vertex_create_buffer_ext(1024 * 1024);

The above code will create a new a new vertex buffer, initially 1MB in size, and store its handle in the variable "v_buff".
