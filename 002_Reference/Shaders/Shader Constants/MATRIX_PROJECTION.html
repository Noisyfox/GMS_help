MATRIX_PROJECTION

The current projection matrix uniform for use in shaders.

Syntax :

MATRIX_PROJECTION;


Returns: Matrix array index


Description

This array index constant holds the index to the current projection matrix, and is one of the available pre-defined matrix uniforms that GameMaker:studio creates for you to use within the shader code editor. The index returned would be used as an array value when calling the gm_Matrices constant within the shader code.


Example :

vec4 pos = gm_Matrices[MATRIX_PROJECTION] * objectspacepos;

The above example is a snippet of GLSL code using the MATRIX_PROJECTION constant.
