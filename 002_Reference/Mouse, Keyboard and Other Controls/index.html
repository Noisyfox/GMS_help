Mouse, Keyboard and Other Controls 鼠标、键盘与其他控制设备

本章节说明了与输入设备相关的全部函数。

最根本的东西，是使一个游戏与一个游戏玩家互动。通过各种各样的手段，用键盘和鼠标的计算机上，或通过在移动设备上的触摸和倾斜。在GameMaker中的标准方法是把动作放入鼠标或键盘事件中，但有时你需要更多的控制，特别是在创建移动设备或HTML5的东西时。你可以使用鼠标和键盘功能按钮被按下或释放以及模拟触摸屏幕上的，也可以有一对特定函数关系到大多数移动设备的倾斜控制。

以下部分是处理 GameMaker:Studio游戏的主要输入设备：

  1. Virtual Keys 虚拟键
  2. Keyboard Input 键盘输入
  3. Mouse Input 鼠标输入
  4. Device Input 设备输入
  5. GamePad Input 手柄输入

在Windows目标平台，您还可以访问传统的操纵杆函数。但是，除非你是一个老游戏的游戏制作与以前的版本移植，建议您使用较新的手柄功能（上图）。操纵杆的函数都可以在这里找到：

  1. Joystick Input 摇杆输入 (Windows Only)

Finally, you also have haptics available to you if you are targeting Android devices. This is covered in the following section:

  1. Immersion Haptics (Android Only)
