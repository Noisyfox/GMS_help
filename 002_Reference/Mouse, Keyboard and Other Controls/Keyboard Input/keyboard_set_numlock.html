keyboard_set_numlock

Set the status of the keypad numberlock.

Syntax :

keyboard_set_numlock(value);


ArgumentDescription
value Set this to true for "on" and false for "off"


Returns : N/A


Description

You can use this function to switch the keypad numberlock on or off (set to true for on, and false for off).

NOTE : This funcionality is only available in the Windows exe builds and will not function on any other device.


Example :

if keyboard_get_numlock()
   {
   keyboard_set_numlock(0);
   }
else
   {
   keyboard_set_numlock(1);
   }

The above example code will get the state of the numberlock key and if it is on (true) it will set it to off (false) and vice-versa.
