Keyboard Input 键盘输入

GameMaker:Studio 中与键盘相关的一些常量和函数，和它怎样可以用来做事情发生在你的游戏中。

当在GameMaker:Studio中处理键盘时，您有各种各样的函数能够被使用，设计到了键盘的不同状态，如按下和释放。也有部分存储键盘按下的字符串或者告知您最近按下的是哪个键，和其他一样允许您完全清除键盘状态。

现在，每个键通常定义一个数字，ASCII代码，你可以直接输入这个数字，这些函数将正常工作... 但是，因为它是有点难以记住这么多数字和关系，他们有与您的键盘，, GameMaker:Studio中有一个系列的最常用的键盘特殊键和一个特殊函数 ord() 从普通字符返回字符类型（字母或数字）。

下面是一个小例子，如何使用ord（）函数：

if keyboard_check(ord('A'))
   {
   hspeed = -5;
   }


因此，上述检查“A”键，如果它被按下，然后它会设置对象的水平速度为-5。请注意，“A”是一个大写字母“A”，并且使用ord（）函数时，键 盘的键检查必须写在引号，作为大写字母。现在，如果你想使用箭头键吗？或者，如果你要修改使用“shift”键的动作吗？那么，该 GameMaker:Studio 有一系列的 vk_ constants（vk_ stands 为虚拟键盘），您可以使用在ord的位置或ASCII码。

if keyboard_check_pressed(vk_tab)
   {
   instance_create(x,y,obj_Menu);
   }


如上代码检查，如“Tab”按下时创建一个“obj_Menu”实例。
