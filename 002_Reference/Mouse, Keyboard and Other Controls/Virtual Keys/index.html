Virtual Keys 虚拟键

这里您将找到如何使用设别虚拟键。

为了您的游戏在移动设备上带有触摸屏的正常工作，您可能需要能够指定的屏幕区域，以应对某些事情时，他们被用户触发。这是通过内部映射的标准键盘键分配一 个“虚拟键”，然后使用键盘产生的事件，他们控制应用程序。这是值得注意的，这些“虚拟键”被映射到屏幕上的位置，而不是以房间计算的位置，并因此，x / y值是绝对的，是根据在显示屏上的大小。这意味着你不必担心使用视角或相对房间坐标，可以简单地利用你的关键的精灵和定义的虚拟键在对象的 Draw GUI Event .


NOTE : 这些函数将不会工作在Windows，也不是MacOS的目标平台！

可用的虚拟按键函数如下:

  1. virtual_key_add
  2. virtual_key_show
  3. virtual_key_hide
  4. virtual_key_delete
