Mouse Input 鼠标输入

本章节关于标准的鼠标输入、

在所有平台上被接受（在移动设备上被接受为一个单一的触摸鼠标输入被接受，如果你需要使用多点触摸，你应该使用 device specific functions) 和一些常量，用于指定按钮被按下。这些常数是在下面的表中所示：

ConstantDescription
mb_leftThe left mouse button
mb_middleThe middle mouse button (this may not be valid for all target platforms)
mb_rightThe right mouse button
mb_noneNo mouse button
mb_anyAny of the mouse buttons



下面的函数存在是为了标准化鼠标按钮控制：

  1. mouse_button
  2. mouse_check_button
  3. mouse_check_button_pressed
  4. mouse_check_button_released
  5. mouse_clear
  6. mouse_last_button
  7. mouse_wheel_up
  8. mouse_wheel_down
  9. mouse_x
  10. mouse_y
