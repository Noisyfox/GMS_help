url_open_full

Opens a URL on a browser, with a specific target and formatting.

在浏览器打开一个URL地址，包含一个目标（如_self这种）和格式。

Syntax 用法:

url_open_full(url, target, options);


ArgumentDescription
url 链接的URL地址（网站地址）。The URL (website address) to link to.
target 打开地址的目标区域（看描述部分）。This is the target area to open the URL in (see description).
options 标准的浏览器选项（看描述部分）。Standard browser options (see description).


Returns : N/A

返回 : N/A

Description 描述

这个函数将会在选定设备的浏览器上打开指定的URL地址，如果你GMS使用的是HTML5模块，同时会打开浏览器。目标区域的参数类似于JavaScript的名字，使用 open()方法时进行赋值，值得注意的是所有的都用 '_self'的形式可能会导致浏览器阻塞，或者询问用户进一步的操作。 "options"相当于JavaScript里的"specs"参数，用来控制窗口或标签显示什么属性（不是所有的浏览器都支持这些属性）。



有效的目标包括：

'_self' : 被点击时在相同的框架内打开链接文档（默认的方式）。
'_blank' : 在新的窗口或标签打开链接文档。
'_parent' : 在父框架内打开链接文档。
'_top' : 破坏框架，在整个窗口或标签打开链接文档。

可用的选项包括 :

'height=[px]' : 窗口的高度，最小值为100。
'left=[px]' : 窗口左边的位置。
'top=[px]' : 窗口上边的位置（只适用于IE）。
'location=[boolean]' : 是否显示地址栏（默认是1）。
'menubar=[boolean]' :是否显示菜单。
'resizable=[boolean]' : 是否窗口大小可调（默认是1）。
'scrollbars=[boolean]' : 是否显示滚动条（默认是1）。
'status=[boolean]' : 是否增加状态栏（默认是1）。
'titlebar=[boolean]' : 是否显示标题栏（默认是1）。可忽略，除非调用的是HTML应用或者对话框。
'toolbar=[boolean]' : 是否显示浏览器工具栏（默认是YES）。

注意: 在 Windows 8 (JS) 目标平台，这个函数也会运行，但是会忽略额外的参数，并激活默认浏览器然后打开URL地址。

This will open the specified URL on the browser of the chosen target device, or, if you are using the HTML5 module, in the currently open browser. The "target" parameter that you specify is the same as the standard JavaScript "name" value when you use the open() method (be aware that all but '_self' may result in the browser blocking, or asking the user if they wish to allow it) and the "options" is the same as the JavaScript "specs" prameter for controlling what properties the new window/tab should display (not all browsers may support all features).

Valid targets are :

_blank _self _parent _top '_self' : Opens the linked document in the same frame as it was clicked (this is default).
'_blank' : Opens the linked document in a new window or tab.
'_parent' : Opens the linked document in the parent frame.
'_top' : Opens the linked document in the full body of the window.

Valid options are :

'height=[px]' : The height of the window, with the minimum value being 100.
'width=[px]' : The width of the window, with the minimum value being 100.
'left=[px]' : The left position of the window.
'top=[px]' : The top position of the window (IE only).
'location=[boolean]' : Whether or not to display the address field (default is 1).
'menubar=[boolean]' : Whether or not to display the menu bar (default is 1).
'resizable=[boolean]' : Whether or not the window is resizable (default is 1).
'scrollbars=[boolean]' : Whether or not to display scroll bars (default is 1).
'status=[boolean]' : Whether or not to add a status bar (default is 1).
'titlebar=[boolean]' : Whether or not to display the title bar. This is ignored unless the calling application is an HTML Application or a trusted dialog box (default is 1);
'toolbar=[boolean]' : Whether or not to display the browser toolbar (default is yes).

NOTE: On the Windows 8 (JS) target platform, this function will work, but additional parameters will be ignored and the default browser will be launched with the given url.


Example :

url_open_full( 'http://yoyogames.com', '_self', 'resizable=0, height=200, scrollbars=0');

This would open the YoYo Games homepage in a new window that can't be resized, has a height of 200 pixels and no scrollbars.

上述代码将在新窗口打 YoYo Games 主页， 大小不可调，高度200像素，没有滚动条。
