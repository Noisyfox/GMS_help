HTML5

These functions are provided specifically for the HTML5 platform.

这些函数是专门为HTML5平台提供的。

HTML5模块有一些是专门设计的函数，被针对HTML5的游戏所使用。应该注意的是，有些可能也适用于其他的目标平台。

The HTML5 module has some functions that are specifically designed to be used by the games that target it, although it should be noted that some of them may also work for other target platforms too. 


可用的HTML5的函数：

  1. url_open
  2. url_open_ext
  3. url_open_full
  4. url_get_domain
  5. browser_height
  6. browser_width
  7. webgl_enabled
  8. clickable_exists
  9. clickable_add
  10. clickable_add_ext
  11. clickable_change
  12. clickable_change_ext
  13. clickable_delete

在HTML5模块中，除了能使用上述函数，在某些特殊的情况下也能使用Javascript变量。比如当html5game文件夹被存储在一个完全不同的地方（URL）时，这个路径往往是作为硬编码存在于Javascript文件中的，并且这个路径是对于javascript被调用的位置的相对路径。而现在可以在调用游戏脚本之前更改g_GameMakerHTML5Dir的值让游戏在一个新定义的位置运行。

Apart from these functions for the HTML5 target module, there is also a special JavaScript variable that can be used on those rare occasions when the contents of the html5game folder are stored in a completely different place (URL). Normally this path is hardcoded in the JavaScript file and is relative to the location from where the javascript is called, but you can set the variable g_GameMakerHTML5Dir before the call to the game script and the game will be run from the newly defined location. For example:

 

<script>var g_GameMakerHTML5Dir = "http://mygamealternatedirectory.com/";</script>
