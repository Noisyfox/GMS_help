clickable_exists

Checks to see if a clickable DOM icon exists.

检查是否存一个可点击的 DOM（文件对象模型，Document Object Model）图标。

Syntax 用法:

clickable_exists(index);


ArgumentDescription
index 需要检查的可点击索引。The index of the clickable to check.


Returns : Boolean

返回 : 布尔值

Description 布尔值

这个函数根据特殊的索引存在与否，返回是否创建了一个可点击的DOM图标。请注意，用来检测的值必须先初始化，否则会产生错误导致 GameMaker:Studio 关闭。

This function returns whether a clickable DOM icon has been created with the specified index exists or not. Please note, that the value used for checking must have been initialised previously or else you will get an error causing GameMaker:Studio to close.


Example :

if !clickable_exists(home_but)
   {
   home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toobar=0, scrollbars=0");
   }

 

上述代码将检测是否存在"home_but"，如果没有检测到，则在游戏运行画布(32, 32)的位置创建一个可点击的DOM图标。图标使用引自纹理页的精灵"spr_MS_Home"，点击将打开具有特殊地址和定义属性的新窗口。

The above code checks the variable "home_but" to see if it already exists, and if it does not it creates a clickable DOM icon at the positon (32, 32) of the page that the game canvas is running on. The icon uses the sprite referenced from the texture page as "spr_MS_Home" and when clicked the icon will open a new window for the specified URL and with the defined properties for that window.
