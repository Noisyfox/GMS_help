date_is_today

Returns whether the given datetime is today or not.

Syntax :

date_is_today(date);


ArgumentDescription
date The datetime to use.


Returns: Boolean


Description

This function will return true if the given datetime value is the day it is being checked on (ie: today), or false otherwise. This can be a handy function for things like Easter Eggs in your games, or for unlocking seasonal content.


Example :

if date_is_today(global.Halloween)
   {
   global.Max_Levels = 200;
   }

The above code will check the datetime stored in the global variable "Halloween" to see if it is today, and if it is it sets another global variable to a new value.
