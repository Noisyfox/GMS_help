Ini files 初始化文件

本章节涉及 ini_file 函数

ini文件的小型化，轻量化的文件，这是兼容大多数平台。他们是存放小件的信息的理想方式，像界面首选项，本地高分数，级别的数据等...

请注意游戏本地化，它是非常重要的，允许字母可以从外部文件读取。这意味着，你应该创建UTF8格式的ini文件中，然后将其添加到GameMaker:Studio中包含的文件，因此，它扩展到运行游戏，用于替代使用默认的ANSI格式的ini文件，当没有之前版本的支持。通过ini函数创造文件，您可以读取和写入到它所有的口音和非罗马字母维持正常。

存在如下ini文件函数。

  1. ini_open
  2. ini_close
  3. ini_write_real
  4. ini_write_string
  5. ini_read_real
  6. ini_read_string
  7. ini_key_exists
  8. ini_section_exists
  9. ini_key_delete
  10. ini_section_delete
  11. ini_open_from_string

WARNING! 这些函数可能无法按预期在 GameMaker:Studio 沙箱中工作！更多信息请参见文件系统限制  File System Limits 。

NOTE :此功能仅限于 Standard Version 的 GameMaker:Studio 及其以上。
