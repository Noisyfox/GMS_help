File System Functions 文件系统函数

本章节的说明处理一般的文件的函数

GameMaker:Studio 里有一些能够以不同方式处理文件的函数：

  1. file_exists
  2. file_delete
  3. file_rename
  4. file_copy
  5. file_find_first
  6. file_find_next
  7. file_find_close
  8. file_attributes

如下函数能够被用于变更文件名：

Note : 这些函数不工作在真实文件上！他们仅处理字符串。

  1. filename_name
  2. filename_path
  3. filename_dir
  4. filename_drive
  5. filename_ext
  6. filename_change_ext

最后，这里有两个特殊函数为获得文件名，它们能被用于载入和保存文件按照文件类型过滤。

Note : 这些函数仅 Windows 可用.

  1. get_open_filename
  2. get_open_filename_ext
  3. get_save_filename
  4. get_save_filename_ext


WARNING! 这些函数可能无法按预期在 GameMaker:Studio 沙箱中工作！更多信息请参见文件系统限制 File System Limits  。

NOTE : 这些函数被限制在标准版 Standard Version 的 GameMaker:Studio 及其以上。
