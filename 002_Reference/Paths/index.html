Paths 路径

本章节涉及基本路径函数。

GameMaker:Studio有一个广泛收集功能和内置的变量为在你的游戏中使用的路径。本节涉及基本的已在路径编辑器 Path Editor 创建的的路径的使用，但更多的功能，允许您使用代码创造自己的路径，他们以不同的方式进行操作。

以下是处理路径的基本函数。

  1. path_start
  2. path_end
  3. path_endaction

以下变量是所有实例创建时内置的，可以被用来获取关于路径信息，或更改某些属性：

  1. path_index
  2. path_position
  3. path_positionprevious
  4. path_speed
  5. path_scale
  6. path_orientation

更高级路径函数请查看以下两个部分：

  1. Path Information 路径信息
  2. Changing Paths 变更路径
