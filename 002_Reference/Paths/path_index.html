path_index

The index of the path the calling instance is following.

Syntax :

path_index;


Returns: N/A.


Description

All resources in GameMaker:Studio have a unique identifying number, and the path_index is a read-only variable that holds the index value for a path resource.


Example :

path_start(choose(pth_enemy1, pth_enemy2, pth_enemy3), 4, 3, 0);

This will make the calling instance follow one of three paths randomly chosen at a speed of 4 pixels per step. It will follow the path relative to its own position, and when it reaches the end it will reverse direction and go back the way it came.
