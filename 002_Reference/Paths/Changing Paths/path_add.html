path_add

Adds an empty path and returns its index.

Syntax :

path_add();


Returns : Real.


Description

With this function you can create a path in GameMaker:Studio without using the path editor. this function will return the indes of the path which should be stored in a variable and used as the reference for that path from then on. Please note that the created path is empty ie: it has no points defined, so you will then have to use the other available functions to add points to the path or be using mp_grids to generate the path. Once you have finsihed using the path, or wish to create a new one and store it's index in the same variable you should first delete the old path with path_delete to prevent memory leaks which can eventually crash your game.

NOTE : This functionality is limited to the Standard Version of GameMaker:Studio and above.


Example :

global.newpath = path_add();

This will create a new path and assign its index to global.newpath.
