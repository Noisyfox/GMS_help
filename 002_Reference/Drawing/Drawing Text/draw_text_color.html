draw_text_color

Draw a string at a given position with a color gradient.

Syntax:

draw_text_color(x, y, string, c1, c2, c3, c4, alpha);


ArgumentDescription
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
c1 The color for the top left of the drawn text.
c2 The color for the top right of the drawn text.
c3 The color for the bottom right of the drawn text.
c4 The color for the bottom left of the drawn text.
alpha The alpha for the text.


Returns : N/A


Description

This function will draw text in a similar way to draw_text only now you can choose the colors to use for coloring the text as well as the alpha value, and these new values will be used instead of the base drawing color and alpha.

NOTE : Gradient blending is not available for the HTML5 target unless WebGL is enabled.


Example :

draw_set_color(c_white); draw_text(100, 100, "Health"); draw_text_color(100, 200, string(health), c_lime, c_lime, c_green, c_reen, 1);

The above code will draw two sections of text on the same line, with the first text being drawn white (as that is the base drawing color) and the second text being drawn with a lime green to normal green gradient.
