2D Primitives 二维图元

本章节为绘制2D图元的所必需的函数。

一个原始的绘图类型是使用点来定义屏幕上显示的是什么。它们可以被制成单一的点，线，或三角形，可以是彩色的，具有透明度混合，甚至可以给出一个纹理，这 意味着它们可以在一个2D游戏中用来创建一些非常令人兴奋的效果！例如，您可以使用多个三角形图元创建一个矩形，然后与一个标志精灵纹理它和在一个实例的 绘制事件中，标志为在进行中的“波浪”，你通过简单的修改点扭曲精灵，形成图元。

NOTE : 这些函数不能用于HTML5模块。

可以在如下页面找到图元信息：

  1. draw_primitive_begin
  2. draw_primitive_begin_texture
  3. draw_primitive_end
  4. draw_vertex
  5. draw_vertex_color
  6. draw_vertex_texture
  7. draw_vertex_texture_color
