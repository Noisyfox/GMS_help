d3d_model_vertex_color

Add vertex (x,y,z) to the model with color values.

Syntax :

d3d_model_vertex_color(ind, x, y, z, col, alpha)


ArgumentDescription
ind The index of the model to add the primitive to.
x The x coordinate of the vertex.
y The y coordinate of the vertex.
z The z coordinate of the vertex.
col The color to blend with vertex.
alpha The alpha value of the vertex.


Returns : N/A


Description

This function defines the position of a vertex of a primitive which is part of a model, and blends the specefied color and alpha with it. The final look of the primitive will depend on the primitive type chosen to draw and the order with which you add the vertexes to it. See d3d_model_primitive_begin for more information. To end and draw the primitive you must call d3d_model_primitive_end.


Example :

model[2] = d3d_model_create();
d3d_model_primitive_begin(model[2], pr_trianglestrip);
d3d_model_vertex_color(model[2], 100, 100, 0, c_red, 1);
d3d_model_vertex_color(model[2], 100, 200, 0, c_red, 1);
d3d_model_vertex_color(model[2], 150, 150, 200, blue, 1);
d3d_model_vertex_color(model[2], 100, 200, 0, c_red, 1);
d3d_model_vertex_color(model[2], 200, 200, 0, c_red, 1);
d3d_model_vertex_color(model[2], 150, 150, 200, blue, 1);
d3d_model_vertex_color(model[2], 200, 200, 0, c_red, 1);
d3d_model_vertex_color(model[2], 100, 100, 0, c_red, 1);
d3d_model_vertex_color(model[2], 150, 150, 200, blue, 1);
d3d_model_vertex_color(model[2], 100, 100, 0, c_red, 1);
d3d_model_vertex_color(model[2], 100, 200, 0, c_red, 1);
d3d_model_vertex_color(model[2], 200, 200, 0, c_red, 1);
d3d_model_primitive_end();

The above code will define a colored tetrahedron and add it to the model.
