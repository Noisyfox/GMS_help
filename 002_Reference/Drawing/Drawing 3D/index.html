Working With 3D 三维工作

本章节详细描述了创建三维游戏的所有函数。

GameMaker:Studio是一个主旨​​在为制作2D（2维）和等距游戏的程序，但有如你所愿如下有些函数是创建3D（3维）图形和3D效果。在开始之前，虽然有几件事你应该明白：

  • 在GameMaker中的3D功能是有限的图形部分。没有其他3D函数的支持。一旦你开始使用3D图形，如果你不使3D工作你可能会得到GameMaker等其他方面的问题。像视野，深度的顺序等等之类的东西.. 需要你有充分的了解如何GameMaker:Studio的工作原理以及如何编写一般的3D。

  • 当您使用的3D函数时，还有一些其他的东西，不可以再使用：
    1. 您不能使用背景和前景在你的房间了，因为他们是填补图像的瓷砖，但在透视投影上无法正常工作。
    2. 您不能再使用普通鼠标位置了，因为鼠标的坐标不会被转化为三维坐标。，但是，仍然可以得到鼠标在屏幕上的（在视野中）的位置，但你必须自己做计算3D位置（或不使用鼠标）。
    3. 你不应该再使用瓷砖，绝大部分的瓷砖将不可能再正确匹配。 碰撞检查的情况下，在房间里依然采用了2-D的位置，所以在3D没有碰撞检测。然而有时候你仍然可以使用，如果你使用的房间为代表扁平的世界（如赛车或FPS游戏），但在其他情况下，你将不得不自己做事情。
  • 所有的3D功能都是通过代码实现，所以你必须对游戏制作语言（GML）有一个正确的理解。

  • 关于3D图形绘制您必须具备一些基本的知识，像透视投影，隐面消除，照明和雾视，类似的东西本帮助不会给太多的解释，并理所当然地认为你是熟悉的条款。

NOTE : 这些函数不工作在HTML5的模块。
