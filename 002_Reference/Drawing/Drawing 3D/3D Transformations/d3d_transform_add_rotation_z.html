d3d_transform_add_rotation_z

Adds a rotation around the z-axis with the indicated amount.

Syntax :

d3d_transform_add_rotation_z(angle)


ArgumentDescription
angle The angle to rotate around the z-axis.


Returns : N/A


Description

This function will rotate the object around the z-axis by the specified amount (in degrees), relative to any previously performed transforms.


Example :

d3d_transform_set_identity();
d3d_transform_add_rotation_z(90);
d3d_draw_block(-50, -50, -50, 50, 50, 50, 100, 100, 100);
d3d_transform_set_identity();

The above code tells GameMaker:Studio to rotate the block 90� along the z-axis (in addition to any previous transforms) before drawing it.
