draw_clear

Clears the entire screen with color.

Syntax:

draw_clear(col);


ArgumentDescription
col The color with which the screen will be cleared


Returns : N/A


Description

This function can be used to draw an opaque color over the entire screen with no alpha blend, and is only for use in the draw event of an instance (it will not show if used in any other event). It can also be useful for clearing surfaces when they are newly created.


Example :

draw_clear(c_blue);

This will clear the screen with the color blue.
