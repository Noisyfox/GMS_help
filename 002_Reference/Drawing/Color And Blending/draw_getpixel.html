draw_getpixel

Returns the color of the pixel displayed at a given coordinate.

Syntax:

draw_getpixel(x,y);


ArgumentDescription
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check


Returns : Real


Description

With this function you can get the color value of any pixel that is currently being drawn on the screen.

NOTE : This function is incredibly slow and so should only be used when absolutely necessary.


Example :

draw_set_color(draw_getpixel(500,200))

The above code will set the drawing color for all text, forms and primitives to the color of the pixel at (500,200) on the game screen.
