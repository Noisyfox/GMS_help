draw_background_tiled_ext

Draws a background at a given position with scaling, blending and alpha, and tiles it to fill the entire room.

在给定的位置，以自定义大小、混合、透明度，绘制背景图块，并填满整个房间。

Syntax 用法:

draw_background_tiled_ext(back, x, y, xscale, yscale, color, alpha);


Argument Description
back 需要绘制的背景。The index of the background to draw.
x 绘制背景的X轴位置。The x coordinate of where to draw the background.
y 绘制背景的Y轴位置。The y coordinate of where to draw the background.
xscale 绘制背景的宽度。The horizontal scaling of the background./td>
yscale 绘制背景的高度。The vertical scaling of the background.
h 背景拉伸占据的区域的高度。The height of the area the stretched background will occupy.
color 混合背景的颜色（正常显示请使用-1或者c_white的形式）。TThe color with which to blend the background (use -1 or c_white to display it normally).
alpha 背景的透明度（从0到1，其中0表示透明，1表示不透明）。The alpha of the background (from 0 to 1 where 0 is transparent and 1 opaque).


Returns : N/A

返回 : N/A

Description 描述

这个函数从给定的坐标开始，根据你定义的大小、颜色混合、透明度（这些属性和 draw_background_ext是一样的），将背景作为图块贴满整个房间。

注意 : 只有WebGL可用时，混合才会对HTML5有效。

This function will take a background and then repeatedly tile it across the whole room, starting from the coordinates that you give in the function and with each tile scaled, color blended and with the alpha that you define (these properties are the same as those used in draw_background_ext).

NOTE : Blending is not available for the HTML5 target unless WebGL is enabled.


Example :

draw_background_tiled_ext(bck_Sky, x, y, 2, 2, c_red, 0.5);

This will draw the background indexed in "bck_Sky" at the instances own x and y position, double it's stored size and tiled in every direction in the room, as well as blended with the color red and partially transparent.

上述代码将索引 "bck_Sky"作为图块，在各自的X/Y位置，双倍于原始图像的大小，混合红色，部分透明，绘制背景且平铺满整个房间。
