draw_background_ext

Draws a background at a given position, with customisable scaling, rotation, blend and alpha.

在给定的位置绘制背景，用户可以自定义大小、旋转、混合和透明度。

Syntax 用法:

draw_background_ext(back, x, y, xscale, yscale, rot, color, alpha);


ArgumentDescription
back 需要绘制的背景。The index of the background to draw.
x 绘制背景的X轴位置。The x coordinate of where to draw the background.
y 绘制背景的Y轴位置。The y coordinate of where to draw the background.
xscale 绘制背景的宽度。The horizontal scaling of the background.
yscale 绘制背景的高度。The vertical scaling of the background.
rot 背景的旋转角度。The rotation of the background.
color 混合背景的颜色（正常显示请使用-1或者c_white的形式）。The color with which to blend the background (use -1 or c_white to display it normally).
alpha 背景的透明度（从0到1，其中0表示透明，1表示不透明）。The alpha of the background (from 0 to 1 where 0 is transparent and 1 opaque).


Returns : N/A

返回 : N/A

Description 描述

这个函数和 draw_background 一样可以绘制背景，不过在绘制时增加额外的选项，可以改变大小、混合、旋转和背景透明度。 更改这些值只是修改绘制的方式，不会影响原始的资源， 你能用任何可用的背景变量代替函数中的参数。

注意 : 只有WebGL可用时，混合才会对HTML5有效。

This function will draw the given background as in the function draw_background but with additional options to change the scale, blending, rotation and alpha of the background being drawn. Changing these values does not modify the resource in any way (only how it is drawn), and you can use any of the available background variables instead of direct values for all the arguments in the function.

NOTE : Blending is not available for the HTML5 target unless WebGL is enabled.


Example 例子:

draw_background_ext(bck_Sky, 0, 0, background_xscale[0], background_yscale[0], 0, c_red, 0.5);

The above code will draw a background at the (0,0) position in the room, with the same scale as the background indexed in background[0], blended red and semi transparent.

上面的代码将会在房间的 (0,0)位置绘制背景，大小和background[0]的一样，混合红色，半透明度。
