room_set_caption

Sets the caption of a given room.

Syntax :

room_set_caption(ind, str);


ArgumentDescription
ind The index of the room to set the caption of.
str The new caption of the room, as a string.


Returns : N/A


Description

This function can be used to set the caption of the given room. This is the same as if you would have set the caption in the room editor itself and can be set to "" (an empty string) for no caption.


Example :

global.rm = room_add();
room_set_caption(global.rm, 'Game Room');

The above code will add a new room to the game and then set the caption for that room.
