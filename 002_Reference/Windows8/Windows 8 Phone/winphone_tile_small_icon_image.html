winphone_tile_small_icon_image

Set the games Iconic Live Tile small icon..

Syntax :

winphone_tile_small_icon_image(fname);


ArgumentDescription
fname The local path to the image to display.


Returns : N/A


Description

This function will take an image and make it the "small" icon of the Iconic type Live Tile. This image must be an included file and not a sprite or a background, and if it is too small or too big for the tile it will be stretched or cropped to fit.


Example :

var tile;
tile = choose("tile1.png", "tile2.png", "tile3.png");
winphone_tile_small_icon_image(tile);

The above code will set the small Iconic Live Tile to be any one of three included files.
