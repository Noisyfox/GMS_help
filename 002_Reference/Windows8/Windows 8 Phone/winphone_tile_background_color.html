winphone_tile_background_color

Set the color of the games Iconic Live Tile background.

Syntax :

winphone_tile_background_color(color);


ArgumentDescription
color the color to use for the background.


Returns : N/A


Description

This function will change the background color of the Iconic type Live Tile. The color can be any one of the color constants, a hex value or the return value for any of the color functions.


Example :

winphone_tile_background_color(make_color_hsv(random(255), 255, 255));

The above code will set the Iconic Live tile background color to a random hue.
