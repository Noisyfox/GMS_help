show_debug_overlay

Show the debug overlay when not in debug mode.

Syntax :

show_debug_overlay(enable);


ArgumentDescription
enable switch on (true) or off (false) the debug overlay.


Returns : N/A


Description

This function can be used to switch on and off the standard debug overlay when testing your game normally. The debug overlay shows a graphic CPU/GPU usage bar in the actual game window itself (along with the current real fps value). This bar is split into sections, with each section being 1/60th of a second. As you can see from the image below, the bar is made up of various colors with white being the GPU left over time, red being the time it takes to run the step event, and yellow the amount of time it takes to draw everything to the screen. In this way, you can see how efficiently your game runs, and get a visual cue as to how it is using the available resources.


Example :

if global.debug
   {
   show_debug_overlay(true);
   }
else
   {
   show_debug_overlay(false);
   }

The above code will toggle the debug on or off depending on the value of a global variable.
