GameMaker:Studio帮助手册

Obsolete Functions 淘汰函数

本节列表了从GameMaker升级到GameMaker:Studio淘汰的函数


由于GMS是专注于快速多平台开发这样的事实,在早期版本的GameMaker有许多功能现在不再支持。大多数是因为游戏设备不允许会造成堵塞而被删除,但一些的移除,因为他们是过时的技术,或因为他们是Windows的特定功能,不再是必要的。

以下的每个页面里的函数已被移除,并简要解释为什么他们被移除:

  1. Triggers 触发器
  2. Registry Functions 注册表函数
  3. CD Functions 音轨函数
  4. Display Functions 显示函数
  5. Splash Functions 飞溅函数
  6. Particle Functions 粒子函数
  7. Sound Functions 声音函数
  8. mPlay Functions 多玩家函数
  9. Message Functions 消息函数
  10. Dynamic Functions 动态函数
  11. Transition Functions 过渡函数
  12. File Functions 文件函数
  13. Miscellaneous Functions 其他函数
