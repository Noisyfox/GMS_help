Obsolete CD Functions 淘汰音轨函数

这些函数是处理windows CD读取的。

Function List :

FunctionDescription
cd_init使 用其它函数前必须调用这个函数。 CD 更换时也要调用（或者只为不时之需）。                                         
cd_present返回值为 CD 当前是否在默认光驱内。
cd_number返回值为 CD 的音轨数。
cd_playing返回值为 CD 是否正在播放。
cd_paused返回值为 CD 是否暂停或者停止播放。
cd_track返回值为当前音轨的编号（ 1 为第一个）。
cd_length返回值为 CD 的总长度，单位为毫秒。
cd_track_length返回值为 CD 音轨 n 的长度，单位为毫秒。
cd_position返回值为在 CD 上的当前位置，单位为毫秒。
cd_track_position返回值为当前播放音轨的位置，单位为毫秒。
cd_play设定 CD 从 first 开始播放到 last 。如果要播放整张 CD ，使用参数 1 和 1000。
cd_stop停止播放。
cd_pause暂停播放。
cd_resume继续播放。
cd_set_position设置在 CD 上的位置，单位为毫秒。
cd_set_track_position设置当前音轨的位置，单位为毫秒。
cd_open_door弹出光驱。
cd_close_door关闭光驱。

如下函数也被删除 :

FunctionDescription
MCI_command这 个函数通过多媒体控制接口（MCI）发送字符串命令到windows多媒体系统。返回值为字符串。你可以使用这个函数控制所有类型的多媒体设备。查看 windows帮助看看如何使用这个命令。例如MCI_command('play cdaudio from 1')播放cd（在你使用其它命令正确的初始化后）。这个函数只提供给高级用户使用。

为什么淘汰这些

这些都是只能在window中使用功能，不再适用于任何其他设备。
