Obsolete Registry Functions 淘汰注册表函数

这些函数处理windows中的注册表。

Function List :

FunctionDescription
registry_write_string用指定的名称和字串在注册表创建一个注册。
registry_write_real用指定的名称和实数在注册表创建一个注册。
registry_write_string_ext用指定的名称和字串在注册表的键建立一个注册。
registry_write_real_ext用指定的名称和实数在注册表的键建立一个注册。
registry_read_string返回指定的名称保存的字串(名称一定要存在，否则会返回空 字串)。
registry_read_real返回指定的名称保存的实数(名称一定要存在，否则会返回0)。
registry_read_string_ext返回键中指定的名称保存的字串(名称一定要存在，否则会返回空字串)。
registry_read_real_ext返回键中指定的名称保存的实数(名称一定要存在，否则 会返回0)。
registry_exists返回该名称是否存在。
registry_exists_ext返回指定的键中指定的名称是否存在。
registry_set_root 设定使用其他时用到的根目录，使用以下数值 : 0 = HKEY_CURRENT_USER, 1 = HKEY_LOCAL_MACHINE, 2 = HKEY_CLASSES_ROOT and 3 = HKEY_USERS.

为什么淘汰这些

只有windows有这些函数，其他设备没有，这些函数在windows上也被认为有严重的安全问题。
