Obsolete Message Functions 淘汰消息函数

这些函数是处理弹出消息和窗口标题的。

Function List :

FunctionDescription
show_message_ext显示一个对话框，里面的信息为字符串，上面还有三个按钮。参数 but1, but2 以及 but3 包含了按钮的文本。空的字符串表示按钮不显示。文本中你可以使用 & 符号来设定下一个用作快捷键的按钮。函数返回值为按下按钮的编号（ 0 为 Esc 键）。
message_background设置上列所有函数的弹出框的背景图。参数 back 必须是某游戏中定义的背景。如果 back 有透明度，信息图像同样也有透明度（ windows2000 或随后的版本才可以）。
message_alpha设置上列所有函数的弹出框为 alpha 半透明。参数 alpha 必须在 0 （完全透明）到 1 （不透明）之间（ windows2000 或随后的版本才可以）。
message_button设置用于弹出框按钮的精灵。参数 spr 必须是一个包含三个图像的精灵，第一个设定未按下及鼠标离开时的按钮，第二个为鼠标进入但没有按下，第三个为按下按钮的图像。
message_text_font设置弹出框里文字的字体。
message_button_font设置弹出框里按钮的字体。
message_input_font设置弹出框里输入框的字体。
message_text_charset这个函数允许您使用指定字符集设置对话框
message_mouse_color设置弹出框里鼠标进入按钮时按钮字体的颜色。
message_input_color设置弹出框里输入框的背景色。
message_caption设置弹出框的标题。参数 show 为是否显示边框，边框显示时用参数 str 设定标题。
message_position设置弹出框在屏幕上的位置。-1，-1为正中
message_size固定弹出框在屏幕上的尺寸。如果你选择1的宽度图片的宽度可用。如果你选择1对高度的计算基于消息的行数。
show_menu显示一个弹出菜单。参数 str 设定菜单文字。菜单由不同的菜单项目（带有垂直条）组成。例如， str = 'menu0|menu1|menu2' 。第一个项目选定时，返回 0 ，以此类推。玩家没有选择项目时，返回默认值。
show_menu_pos同上个函数一样显示弹出菜单，不过这个函数还指定菜单在屏幕上的位置。
get_color获取玩家选哪个颜色。参数 defcol 为默认颜色。如果玩家按下 Cancel 键，返回值为 -1 。
get_directory获取目录。参数 dname 为默认名。如果玩家按下 Cancel 键，返回值为空字符串。
get_directory_alt可选择的方法获得目录。参数 capt 为显示的标题。参数 root 为显示根目录。使用空字符串来显示整个目录。如果玩家按下 Cancel 键，返回值为空字符串。
show_score是否在窗口标题中显示得分。
show_lives是否在窗口标题中显示生命数量。
show_health是否在窗口标题中显示健康值。
caption_score用于得分的标题。
caption_lives用于生命数量的标题。
caption_health用于健康值得标题 。
highscore_show显示最高分表格。参数 numb 为新分数。如果分数可以添加到列表，玩家可以输入名字
highscore_set_background设定使用的背景图。参数 back 必须为背景资源里的索引名。
highscore_set_border设置最高分表是否有边框。
highscore_set_font设置表格中使用的字体。
highscore_set_colors设置背景使用的颜色，表格的新条目的颜色，以及其他条目的颜色。
highscore_set_strings显示最高分表格时改变不同的默认字符串。
highscore_show_ext显示带大量选项的最高分表格（也可以用先前大量的函数做）。
highscore_add_current添加当前分数到最高分列表。玩家可以输入一个名字。

以下常量也被删除：

  • text_type
  • button_type
  • input_type

为什么淘汰这些

 这些都是只能在window中使用功能，不再适用于任何其他设备。特别是显示消息框的函数。

Note : 而高分榜函数现在已经过时了,内置高分榜仍然存在,可以用…你将必须通过自己的代码显示它们。
