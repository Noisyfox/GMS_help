Obsolete Sound Functions 淘汰声音函数

这些函数是处理音效和3D声音的。

Function List :

FunctionDescription
sound_effect_set设 置指定声音的声效。参数 effect 可以为设定好的值。                                                                                              
sound_effect_chorus为指定的声音设置和声效果的参数
sound_effect_echo为指定的声音设置回声效果的参数。
sound_effect_flanger为指定的声音设置去噪效果的参数。
sound_effect_gargle为指定的声音设置漱口声效果的参数。
sound_effect_reverb为指定的声音设置混响效果的参数。
sound_effect_compressor为指定的声音设置压缩器的参数。
sound_effect_equalizer为指定的声音设置均衡器的参数。
sound_3d_set_sound_position设置声音的位置，空间中关系到听众的指定位置。
sound_3d_set_sound_velocity设置声音的速率，空间中指定声音速率的向量值。
sound_3d_set_sound_distance设置声音不再增加响度的最小距离，还有声音不再能听到的最大距离。
sound_3d_set_sound_cone 正常的声音在指定距离的所有的方向上有相同的振幅。
sound_get_preload返回值为指定索引号的声音是否设置预加载。
sound_restore恢复当前播放的指定声音的内存。
sound_discard释放指定声音使用的内存。
sound_background_tempo改变背景音乐的速度（midi格式）。
sound_set_search_directory设置 direct 音乐格式的路径目录。
sound_pan 改变指定声音的声道（ -1 为左， 0 为中间， 1 为右）。

以下常量也被删除 :

  • se_none
  • se_chorus
  • se_echo
  • se_flanger
  • se_gargle
  • se_reverb
  • se_compressor
  • se_equalizer

为什么淘汰这些

这些函数只能在windows上运行，其他设备不支持.
