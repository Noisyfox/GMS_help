part_type_color1

Sets a particle type to be a single color throughout its lifespan.

Syntax :

part_type_color1(ind, color1);


ArgumentDescription
ind The index of the particle type to change.
color1 The single color to make the particle type.


Returns : N/A


Description

This function is used to set a particle type to be a single color for the total duration of the lifetime of each individual particle.


Example :

part_type_color1(global.Snow_Part, c_white);

The above code will set all particles created of the particle type indexed in the global variable "Snow_Part" to be white only.
