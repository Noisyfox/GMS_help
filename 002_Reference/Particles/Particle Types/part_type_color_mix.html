part_type_color_mix

Sets a particle type to be any blend of two given colors.

Syntax :

part_type_color_mix(ind, color1, color2);


ArgumentDescription
ind The index of the particle type to change.
color1 The first color the blend will take from.
color2 The second color the blend will take from.


Returns : N/A


Description

With this function you can set the given particle type to be a random blend of two colors.


Example :

part_type_color_mix(global.P_Damage, c_red, c_yellow);

The above code will set the color for each particle emitted of the particle type indexed in the global variable "P_Damage" to be a random mix between the colors red and yellow.
