exp

The exponential function.

Syntax :

exp(n);


ArgumentDescription
n The power of e.


Returns : Real


Description

This is the function power(e,n), where e is approximately 2.718281828 (also known as Euler's Number), and n is the number of times it should be multiplied by itself. This basicaly says "how much growth do I get after after n units of time (and 100% continuous growth)?". So exp(3) would return 20.08, which means that after 3 time units we have 20.08 times the amount we started with.


Example :

val = exp(2);

This will set val to e^2, around 7.4.
