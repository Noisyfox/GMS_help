sin

Returns the sine of val (val in radians).

Syntax :

sin(val);


ArgumentDescription
val The angle (in radians) to return the sine of.


Returns : Real


Description

In a right angled triangle sin(val) = Opposite / Hypotenuse where val is one of the three angles.


Example :

val = sin( pi/2 );

This will set val to 1.
