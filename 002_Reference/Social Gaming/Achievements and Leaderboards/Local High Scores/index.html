Local High Scores 本地高分榜

本节涉及的一些特殊的函数，用于创建一个本地的高分表。

为了帮助您在您游戏中创建和维护高分，GameMaker:Studio 创建一个全局性的有10个位置的高分阵列，你可以访问，添加和更改，以创建自己的自定义高分列表，这没有太多的大惊小怪的。这是本地游戏的功能，是工作在所有平台上的，所以你可以很容易地存储和维护一个基本的高评分表，与文件处理函数 File Handling Functions 一起使用这些函数。

如下函数是关于整合本地高分榜的。

  1. highscore_add
  2. highscore_name
  3. highscore_value
  4. highscore_clear

GameMaker:Studio 还包括一个函数，在屏幕上的绘制的高分列表，但是它是非常基本的，建议您只在调试时使用，因为自己制作的高分表很容易的，这样做将给予你更多的控制它是如何显示的：

  1. draw_highscore
