facebook_send_invite

Send an invite message to friends.

Syntax :

facebook_send_invite(titlestring, msg, picture_url, redirect_url);


ArgumentDescription
titlestring The title of the dialog (a string)
msg The message which appears in the dialog (a string).
picture_url the url of the image to include in the dialog.
redirect_url The �SiteURL?of your app on the Facebook Developers app settings page.


Returns : N/A


Description

With this function you can get your game to display a facebook �apprequests?dialog, which will allow your player to select friends from a list and send them a game invite message.

NOTE : The "SiteURL" is ignored on the iOS and Android modules and is only required on the HTML5 module if you are not hosting the application on the "SiteURL" (ie: you are testing locally).


Example :

if mouse_check_button_pressed(mb_left)
   {
   facebook_send_invite("Catch The Haggis!", "Come and play Catch The Haggis with me...", "http://MacSweeneygames.com/Clown.jpg", "");
   }

The above code will display an "apprequest" dialogue where you can select friends to invite to play the game.
