audio_stop_all

Stops the indicated sound.

Syntax :

audio_stop_all(index);


Returns : N/A


Description

This function will stop all sounds that are currently playing.

NOTE : This function will only stop those sounds that have been flagged as "Normal Sound" when loaded into GameMaker:Studio. If the sound is flagged as "Background Music" this function will not work and you should use the function audio_stop_music.


Example :

if !global.SFX
   {
   audio_stop_all();
   }

The above code checks the global variable "SFX" and if it returns false, it will stop all sounds that are currently playing.
