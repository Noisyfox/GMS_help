audio_get_type

Returns the type of audio for the given sound.

Syntax :

audio_get_type(index);


ArgumentDescription
index The index of the sound to check.


Returns : N/A


Description

When adding audio to GameMaker:Studio it can be either sound or background music, with different functions being applicable for either of the two types. If you need to know what type of sound an index points to you can use this function which will return 1 for music, 0 for a basic sound, and -1 if there is any error or the index does not point to a valid sound resource.


Example :

type = audio_get_type(ss);

The above code checks the type of audio indexed in the variable "ss" and stores the returned value in the variable "type".
