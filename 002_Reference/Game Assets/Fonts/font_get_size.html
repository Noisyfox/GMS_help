font_get_size

Returns the size of a given font.

Syntax :

font_get_size(ind);


ArgumentDescription
ind Index of the font to get the size of.


Returns: Real


Description

With this function you can get the size of any font resource, which is the point value shown by the font resource dialogue.


Example :

sz = font_get_size(font0);

This will get the size of the fonty indexed by the "font0" variable and store it in the variable "sz".
