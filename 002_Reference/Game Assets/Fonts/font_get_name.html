font_get_name

Returns the resource name of a given font.

Syntax :

font_get_name(ind);


ArgumentDescription
ind Index of the font to check.


Returns : String


Description

This function can be used to get the name that was given to the font when it was added as an asset to the GameMaker:Studio resource tree. The return value is a string and not an index.


Example :

fnt_Name = font_get_name(font0);

The above code will get the name of a font resource as it appears in the resource tree and store it as a string in the variable "fnt_Name".
