background_color

This variable can be changed to change the current room's background color.

Syntax:

background_color


Returns : Real


Description

When in a room, it can have no background image selected for any of it's background_index, in which case you should set a background color with this variable and make sure that the background_showcolor variable is set to true so it can be seen. Having a room with no background image and no background color will result in graphical artefacts being drawn on the screen and is not recommended.


Example :

if background_index[0] = -1
   {
   background_color= c_black;
   background_showcolor = true;
   }

The above code checks to see if background0 has an image assigned to it and if not it sets the background color to black and flags it to be shown.
