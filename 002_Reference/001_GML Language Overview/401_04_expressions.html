Expressions 表达式

本章节讨论不同的数学表达式

表达式可以是数字（比如：4、3、5、6、等等），十六进制数字 $ 打头的格式（ 比如 $00FFAA），单引号或双引号括起来的字符串（比如"Hello"或'hello'）或者其他更复杂的表达式。对于表达式，存在下面一些运算符（按照优先级 ):
 

赋值运算符
= - 用于赋值到变量。注意这也用于在GMS中比较变量，您可以在下面的例子和其他人的代码中看到。然而在更老的版本中你应该使用==作为比较，=作为赋值。例子请见如下：

a = 12; speed = 5; val = (old_val + 5);

 

逻辑运算符
 

&&, ||, ^^ (and, or and xor) - 结合布尔值返回 true 或 false。如下例子中的表达式如果为 true 则执行代码。

if (a == b && c == d) {do something...}// and
if (a == b || c ==d) {do something...}// or
if (a == b ^^ c == d) {do something...} // xor

 


比较运算符
<, <=, ==, !=, >, >= - 这些都是比较，只能返回 true 或 false的结果（true为1，false为0）。例子如下：

if (a < b) {do something...}
if (a != b) {do something...}

 


位运算
|, &, ^, <<, >> : 您可以执行位运算。| 等于 或, & 等于 并, ^ 等于 异或, << 等于 左移, >> 等于 右移. 例子如下:

x = (x & $ffffffe0) + 32;
if (y ^ $1f) > 0 {do something...};

参考百科位运算：http://baike.baidu.com/view/379209.htm?fr=aladdin

 


算术运算符
+, -, *, / - 加减乘除，例子如下：

c = a * b;
str = a + "world";

 


递增/递减
++, -- - 加1或减1。值得注意的是在值的前面和后面结果不同，例子如下：
  • ++a 加后返回结果。
  • a++ 加前返回结果再执行加。

例子如下:

var a = 1;
show_debug_message(string(a++));
show_debug_message(string(++a));

 debug 输出结果为 1 和 3. 另外例子如下:

for (i = 0; i < 10; i++) {do something...};
if hit == true score--;

注意：在YYC目标平台上，这些表达式从左到右的求值，而其它平台他们是从右到左的求值，这意味着如下表达式：

val = max(num,++num,num++);

根据不同的目标平台会得到不同的结果。


整除和取余
div, mod (%) - 整除和取余，整除是两个整型相除，截尾取整数部分。取余是两个整型数相除取余数部分，余数符号与被除数符号相同，示例如下： 

a = 3 div 2  //3除以2，结果为1余1，结果取整数部分后为1，故a为1

a = 6 div 3  //6除以3，结果为2余0，结果取整数部分后为2，故a为2

a = 3 mod 2 //3除以2，结果为1余1，余数为1，故a为1

a = 4 & 5 //4除以5，结果为0余4，余数为4，故a为4 

secs = time mod 60;
time_str = string(time div 60);

 

另外还有如下一元运算符：Also, the following unary operators exist:

  • ! (not) : 否, true 转换为 false ， false 转换为 true
  • - : 否定后面的数值
  • ~ : 否定后面的数值进行二进制位取反（1变0，0变1）

 

总结

作为表达式的值，可以使用数字，变量，或函数返回值，也可以置于表达式之间。全部的操作符可以使用在实数上，比较运算可以用于字符串，“+”可以连接字符串。

在一个表达式中使用多个运算符时，为了保证一定运算的次序，使用括号隔开将非常重要，如果没有这样处理，对于不同的目标平台将得到不同的运算结果。比如下面的代码：

a = b == c || d;

由于没有明确的显示，不同的目标编译器将出现不同顺序的运算，得到无法理解的结果。为了避免这个，请使用括号隔开各个部分：像这样：

a = ( b == c || d) ; //better

a = ((b == c) || d ); //best

Note : 与某些语言相反，两个参数之间总在运算，一个参数决定着另外一个参数。
 

整合例子如下:
 

{
x = 23;
color = $FFAA00;
str = 'hello world';
y += 5;
x *= y;
x = y << 2;
x = 23 * ((2 + 4) / sin(y));
str = 'hello' + " world";
b = (x < 5) && !(x==2 || x==4);
}
