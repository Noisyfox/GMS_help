The "If" Statement

这个章节描述了如何使用 "if" 语句

简单的"if" 表达形式如下 :

if (<expression>) <statement>


或者有更复杂的形式 "if... else..." ：

if (<expression>) <statement> else <statement>


 在这种表达形式下，如果值是<=0 (false)则执行else，如果值(true) 则继续执行后面代码，使用 if 语句时的好习惯是给每个语句放在新的一行，在最后写入结束格式。

if (<expression>)
   {
   <statement>
   ...
   <statement>
   }
else
   {
   <statement>
   }


作为一个小例子,下面的代码将移动在房间中的实例向位置x = 200：

if (x<200)
   {
   x += 4;
   }
else
   {
   x -= 4;
   }
