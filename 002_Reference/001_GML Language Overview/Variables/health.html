health 健康

全局变量用于保存玩家健康值

Syntax :

health;


Returns : Real


Description

这个变量是全局变量用于保存记录玩家健康。记住这是不一定总是如此——只是因为它叫做“健康”并不意味着它必须被用于健康…它可以容纳任何实际或整数价值你希望你的游戏。


Example :

if place_meeting(x, y, obj_Enemy)
   {
   health -= 10;
   if health <= 0 instance_destroy();
   }

上面的代码检查当前实例是否存在碰撞的实例对象索引在“obj_Enemy”,如果有,它去掉了10分从全局健康变量,然后检查这个值是否小于或等于0。如果它是则实例毁灭自己。
