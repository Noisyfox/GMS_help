Functions 函数

这个章节描述了什么是 function 

函数的一般定义是这样的:一个函数有输入和输出,输出与输入有潜在关系。在GMS中你使用函数作为 program 的一部分，使得游戏发生事件并且有大量的GML(GameMaker语言)功能可用，全部可以在帮助的 Reference 章节中查看。

在GML函数形式函数的名称,接在后面的括号之间的输入参数和逗号分隔(如果函数没有输入参数就使用括号)。这是一个概括的结构函数:

<function>(<arg0>, <arg1> ,... <arg15>);


一般有两种类型的函数——首先,有海量的内置函数用于控制所有方面的游戏,其次,还可以用任何自定义 scripts 在你的游戏作函数(但不总是)。

一些函数的返回值,可以被用在表达式,而其他只是执行命令,你应该注意,不可能使用一个函数作为左边侧的一项任务。例如,你不能写:

instance_nearest(x, y, obj).speed = 0;


作为返回值的表达式是一个实数,所以必须括在括号中使用这种方法 (见 Addressing Variables In Other Instances ), 所以上述代码实际上应该写成:

(instance_nearest(x,y,obj)).speed = 0;
