Programs 程序

在这里您可以找到使用 GML 的基本结构。

程序包括一组指令集，称为语句，然后在您的游戏中通过GameMaker:Studio 解释执行。这就好比2与2获得4一样，或者复杂到敌人血低到一定时会逃跑。实际的程序结构差别很大，取决于函数的使用目的，基本的结构参看如下：

{
<statement>;
<statement>;
...
}

程序必须开始是'{' 结尾是 '}'，在符号与语句之间要使用 ';' 分隔。更多的表现形式如下面，这是在 GameMaker:Studio Script Editor 创建的脚本:
