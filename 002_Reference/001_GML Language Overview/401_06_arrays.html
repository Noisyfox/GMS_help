Arrays 数组

本章节是处理数组,特殊的变量,它可以包含多个值。

第一次看到数组会有些困惑，但是它们是非常有用的，是制作游戏的重要组成部分。为什么？嗯,这里有一些事情,是不可能没有数组的 -

  • Menus. 一个数组或两个可以让你更容易创造一个良好的菜单系统。
  • RPGs. 制作RPG数组是必要的，因为可以替代大量变量,你只需几行，您可以在任何时候重来。
  • Card Games. 有利于跟踪卡片和手,甚至还会洗牌!
  • High scores and other statistics. 跟踪一个数组比追踪多个变量更容易。

那只是冰山一角作为数组是一个最基本的和有用的编程工具可以使用，你会很惊讶的发现，在应用程序都有!让我们找出更多关于他们然后……

1D Arrays

好吧，对上面提到的那些事情我们可以使用一个数组，但什么是一个数组？什么样子？嗯,这样...

array[0] = 1.25;


这是一个一维数组(一维),我们现在要分解的部分看到它意味着什么：

array


这是数组的名字。就像任何其他的变量,它可以是任何你想到的，从简单的“a”到“mymumscow”。

[0]


这是数组中的位置，我们正在检查或改变。你看，一个数组基本上是一个容器与一些空间来存储值，每个位置都有一个特定的容器号码来识别它,这正是我们将在[]填入的。值得注意的是，一个数组总是始于0和不能为负数！现在让我们扩大我们的数组包含不同的位置...

array[2] = 0;
array[1] = 0;
array[0] = 0;


我 们现在包含三个位置数组(0、1和2),我们把数组初始化到0。这是什么意思?嗯，一个数组必须初始化，我们可以使用之前不然GMS将给我们一 个错误。初始化一个数组只是意味着我们给每个位置的数组初始值在准备它用于其它对象或代码时。要记住这是重要的，因为它意味着你必须做一定数量的规划在使 用数组，但它是很容易使用一个重复循环初始化的...

var i;
i = 9;
repeat(10)
   {
   array[i] = 0;
   i -= 1;
   }


这个简单的代码将初始化十个位置的数组(0 - 9)为0,在每个位置的数组包含值0。你 会发现，阵列已向后初始化，首先定义的最后一个值。这不是严格必要的，但最佳的方式来做到这一点，因为它会保留在记忆体的空间中，这是确切的大小的数组， 而如果你从0向上初始化一个数组，存储器已被重新分配给每个额外的附加值。速度差较小的阵列中可以忽略不计，但较大的应尽可能地以这种方式进行优化。

但是，如果我们想用不同的值来初始化数组，每个位置？好吧，我们有手动键入每个位置，我们自己，而是有一个很不错的技巧，以帮助我们保持跟踪那里太多的东西！

count = 3;
array[count] = "you?"
count -= 1;
array[count] = "are "
count -= 1;
array[count] = "How "
count -= 1;
array[count] = "Hello!"
count -= 1;


正如您可以看到的,我们没有使用任何数据在实际的数组,而一个变量通过值开始数数。这有两个好处,一,我们不必担心拼写错误或写出错误的数组位置,二,我们已经在变量“count”位置的数量,该数组包含,然后在对象其他位置可以使用。这非常有用!

完成之后我们如何使用数组为实际的东西?完全一样,我们将使用一个正常的变量,如以下示例所示：

total = array[0] + array[5]; //Add two array values together

if array[9] = 10 //Check an array value
   {
   //do something
   }

draw_text(32, 32, array[3]); //draw an array value


因为数组是连续编号,这意味着您可以循环执行额外的动作也他们像我们那样对它进行初始化那样：

var i, total;
i = 0;
repeat(10)
   {
   total += array[i];
   draw_text(32, 32 + (i * 32), array[i]);
   i += 1;
   }
draw_text(32, 32 + (i * 32), total);


上面的代码将添加所有的值在我们的数组中,绘制它们,并在最后绘制总值。

2D Arrays

现在我们知道普通的数组是什么样子,让我们看看一个2维(二维)数组

array[0,0] = 5;


像之前一样,每个数值指向一个数组内的位置,只有这一次每个位置有一个“a”和“b”坐标。可以认为这是一个额外的维在我们的容器中,因为它现在已经高度和宽度而1维数组只有高度。这是一个扩展的例子

array[1,2] = 1;
array[1,1] = "hello";
array[1,0] = 55.5;
array[0,2] = sprite_index;
array[0,1] = "world";
array[0,0] = -67.89;


二 维数组需要在使用前初始化,类似一个一维数组,可以容纳真正的数字、字符串和常量,就像任何其他变量,使它们适合任何游戏,需要存储大量的数据 时是一个方便的方法(记住,你可以简单遍历一个数组)。这是最后一个例子,这可能被用于实际的游戏……说是你想根据随机值产生四种不同的敌人在四个不同的 点在你的游戏中。嗯,我们可以用一个二维数组来做到这一点用来保存的代码并读取代码。

首先我们应该初始化数组,我们将使用在“controller”对象的创建事件中。

enemy[3,2] = 448; //y position
enemy[3,1] = 32; //x position
enemy[3,0] = obj_Slime; //Object
enemy[2,2] = 448;
enemy[2,1] = 608;
enemy[2,0] = obj_Skeleton;
enemy[1,2] = 32;
enemy[1,1] = 608;
enemy[1,0] = obj_Knight;
enemy[0,2] = 32;
enemy[0,1] = 32;
enemy[0,0] = obj_Ogre;


我们现在有生成的对象实例及其对应房间的x和y坐标,都储存在我们的数组中。现在这个可以使用在另一个事件中控制对象(例如,警报或键盘按键事件):

var i;
i = irandom(3); //get a random number from 0 to 4
instance_create(enemy[i,1], enemy[i,2], enemy[i,0]); //Use the array to create the object


这个简短的代码将生成一个随机的敌人在游戏房间,它使用更少的代码远比一个“if / then / else”结构或甚至是一个“switch”,以数组初始化一起创建事件很容易编辑和改变这些值，而不是硬编码到其余的对象代码。

Advanced Array Functionality 高级数组函数

就像普通变量，您可以通过脚本使用，然后调用脚本返回数据给实例。要做到这一点，你只需到指定的数组变量（无需对应每个单独项，也不括号[]），这些值将通过脚本被复制到一个临时数组。请注意这里临时这个词使用！实际上，你是不是传递数组本身到脚本（变量），而是要求该脚本创建一个副本，这个数组。这意味着，如果你想以这种方式改变任何数组值，你必须从脚本返回数组。

NOTE: 由于内部工作方式，数组传递到脚本可能会影响性能，尤其是如果该数组是非常大的。因此，小心使用这个功能！

作为一个例子，考虑下面的代码。首先，我们创建我们要使用的数组，然后我们这个数组传递给脚本！

for (var i = 9; i > -1; i--;)
   {
   a[i] = i;
   }

scr_Return_Array(a);


脚本本身有些相像：

for (var i = 9; i > -1; i--;)
   {
   a[i] = i * 100;
   }


  现在，你会想到最后的数组来保存值900，800，700等..但是，这不会是这种情况，因为我们没有从脚本返回的数组，因此所有我们改变临时的副本，当我们通过数组作为参数到脚本创建。要纠正这一点，我们应该如下这样格式化代码：

for (var i = 9; i > -1; i--;)
   {
   a[i] = i;
   }

a = scr_Return_Array(a);


脚本应该看起来像这样：

for (var i = 9; i > -1; i--;)
   {
   argument0[i] = i * 100;
   }

return argument0;


您也可以简单地删除数组"re-assigning"变量，它定义为单个值。这将腾出该数组中的所有项和值相关联的内存。例如：

//Create the array for (var i = 9; i > -1; i--;)
   {
   a[i] = i;
   }
//Delete the array a = 0;
