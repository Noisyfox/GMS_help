The "While" Statement

这个章节描述了如何使用 "while"

  "while" 表达形式如下 :

while (<expression>) <statement>


只要表达式为true,语句(也可以是一个代码块)被执行。小心你的while循环!你可以很容易地导致无限循环,在这种情况下你的游戏将挂起,不响应任何用户输入了。下面你可以找到一个例子,一个典型的方式使用“while”:

{
while (!place_free(x, y))
   {
   x = random(room_width);
   y = random(room_height);
   }
}


上面的程序试图将当前对象移到一个自由的位置(这样的行动也可查阅 move an object to a random position).
