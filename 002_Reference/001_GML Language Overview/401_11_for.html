The "For" Statement

这个章节描述了如何使用 "for".

"for" 表达形式如下 :

for (<statement1> ; <expression> ;<statement2>) <statement3>


这种工作方式如下,首先执行语句1,然后在计算表达式,如果这是真的,声明3是执行。然后声明2,然后再在计算表达式。这个循环将继续直到表达式是发现是假的。

现在,这听起来很复杂,但写的时候你应该理解为:

  • 第一个语句初始化循环。
  • 表达式测试循环是否应该结束。
  • 语句2是“步”声明,我们要到下一个循环的判断。

这非常适合用于做重复性的任务,在任何其他方式涉及多个行代码,通常用作计数器为判断数组,或绘制。下面的代码示例演示了一个典型的使用这种类型的声明:

{
var i;
for (i = 0; i < 10; i += 1)
   {
   draw_text(32, 32 + (i * 32), string(i) + ". "+ string(scr[i]));
   }
}


上面的代码初始化 for循环,从0开始计数,直到9,然后使用循环值的“i”把存储在数组中的值绘制在屏幕。注意“for”循环变量的“i”不仅是用来遍历该数组,还作为一个绘制变量告诉GMS在房间哪里绘制。这给与灵活性设计其中的一个非常重要的结构。
