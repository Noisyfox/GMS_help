The Game Window 游戏窗口

章节是关于游戏窗口的全部函数.

实际游戏你创建发生在一个窗口，此窗口中有许多的属性，如是否有边框，是否是全屏等，通常情况下，这些值都设置在  Windows Tab 的全局游戏设定 Global game Settings 但你可以在游戏中改变它们。

NOTE : 这些函数仅适用于Windows，MacOS和HTML5的目标模块，任何其他设备上可能无法正常工作。
