view_visible

决定屏幕上是否显示特定的视野。Whether the particular view is visible on the screen.

Syntax 用法:

view_visible[0...7]


Returns 返回: Boolean 布尔值


Description 描述

这个变量被用来找出特定的视野当前可见或者不可见。你能设置变量的“true”或者“false”来影响视野的开启或者关闭。

This variable can be used to find out if a particular view is currently visible or not. You can also set this variable to effectively turn "on" or "off" a view by setting the value to to true (visible) or false (invisible).


Example :

if !view_visible[0]
   {
   view_visible[0] = true;
   }

 

The above code checks to see if view[0] is visible or not and if it is not, it then sets it to be visible.

上述代码检测view[0]是否可见，如不可见，则让它显示出来。
