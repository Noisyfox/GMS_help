view_xview

在房间里，视野的X位置。 X position of the view in the room.

Syntax（用法）

view_xview[0...7]


Returns 返回: Real 实数


Description 描述

这个变量可以用来获取或设置视野的x轴位置。改变这个值，你可以将视野设置到一个新的位置，（0,0）在房间（room）的左上角，您也可以参考这个变量以确保实例总是绘制在视野内一个相对的位置（见下面的示例代码）。该值也可以是负的，显示的区域在房间（room）的外面，在此情况下，可以使用窗口颜色（这种颜色可以用window_set_color设定）。

This variable can be used to get or to set the x axis position for the view in the room. By changing this value you can set the view to a new position, with (0,0) being the upper left corner of the room, or you can reference this variable to ensure that an instance is always drawn at a relative position to the view (see example code below). This value can also be negative and so show an area that is outside of the room, in which case the window color will be used (this color can be set with window_set_color).





Example 例子

draw_text(view_xview[0] + 32, view_yview[0] + 32, "Score : " + string(score));

上面的代码将在视野view[0]的X坐标和Y坐标一个相对的位置绘制得分。

The above code will draw the score at a position relative to the x and y coordinates of view[0].
