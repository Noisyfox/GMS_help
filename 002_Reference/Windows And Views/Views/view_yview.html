view_yview

view_yview

房间里视里的Y坐标。Y position of the view in the room.

Syntax 用法:

view_yview[0...7]


Returns 返回: Real 实数


Description 描述

这个变量用来获取或设置视野的Y轴位置。 通过改变这个值，可能将视野设置到一个新的位置，比如 (0,0) 是房间左上角的位置，或者你也可以参照这个变量以确保实例总是绘制在视野的一个相对的位置（后面的例子会说到）。这个值也可以是负的，然后它会在房间外面显示一块区域，这种情况下将使用到窗体颜色（使用window_set_color设置）。

This variable can be used to get or to set the y axis position for the view in the room. By changing this value you can set the view to a new position, with (0,0) being the upper left corner of the room, or you can reference this variable to ensure that an instance is always drawn at a relative position to the view (see example code below). This value can also be negative and so show an area that is outside of the room, in which case the window color will be used (this color can be set with window_set_color).




Example 例子:

draw_text(view_xview[0] + 32, view_yview[0] + 32, "Score : " + string(score));

上面的代码将在视野view[0]的X坐标和Y坐标一个相对的位置绘制得分。

The above code will draw the score at a position relative to the x and y coordinates of view[0].
