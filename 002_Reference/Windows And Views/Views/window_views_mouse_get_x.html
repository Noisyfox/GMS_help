window_views_mouse_get_x

相对于窗口内给定的所有激活的视野，返回鼠标位置的X坐标。Returns the x-coordinate of the mouse with respect to the views.

Syntax 用法:

window_views_mouse_get_x();


Returns 返回: Real 实数


Description 描述

这个函数将返回相对于所有激活视野的鼠标位置的X坐标，和mouse_x返回相同的坐标值。

This function returns the x-coordinate of the mouse with respect to all the active views and returbns the same value mouse_x.


Example 例子:

myx = window_views_mouse_get_x();

This would set myx to the x coordinate of the mouse in the window relative to all active views.

上述代码将设置myx变量为窗口内鼠标位置X坐标，这个鼠标位置坐标相对于所有激活的视野。
