view_object

对象的实例必须保持在视野里可见。Object whose instance must remain visible in the view.

Syntax 用法:

view_object[0...7]


Returns 返回: Real 实数


Description 描述

有了这个变量，你能获得或设置对象让其一直保持在指定的视野（从0到7）中，如果对象有多个实例，只有一个会跟随视野（通常是第一个创建的，但不能完全保证）。你也可以为这个变量指定单个实例的id，这种情况下，这个特别的实例将会跟随。如需要阻止任何视野跟随，你可以设置变量为一个特殊的键noone。

With this variable you can get (or set) the object that has to be kept in the specified view (from 0 to 7) at all times. If there are multiple instances of this object, only one is followed (usually the first one created, but this is not gauranteed). You can also assign a single instance id to this variable, in which case that particular instance is followed. To prevent any view following, you can set this variable using the special keyword noone.



注意 : 在视野开始跟随之前，你可以使用view_hborder 和 view_vborder 函数设置实例或对象的移动距度。

NOTE : You can set the distance that an instance or object can move before the view starts following using the view_hborder and view_vborder functions.

Example :

if instance_exists(obj_Player)
   {
   view_object[0] = obj_player;
   }

上述代码将检查是否存在对象"obj_Player"的实例，如果存在，则设置视野view[0]跟随这个实例。

The above code checks to see if an instance of the object "obj_Player" exists and if it does, it sets view[0] to follow that instance.

更正说明 : 官方文档中“view_follow[0] = obj_player;”应该写错了，正确的函数是"view_object"，上述代码中已经作了更正。
