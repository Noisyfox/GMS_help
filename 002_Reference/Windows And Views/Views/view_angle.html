view_angle

视野的旋转角。Rotation angle used for the view in the room.

Syntax 用法:

view_angle[0...7]


Returns 返回: Real 实数

更正说明: 原返回值是布尔值，Globalvar提供资料“Rotation angle used for the view in the room (counter-clockwise in degrees).”表明这个变量应该是一个实数而不是布尔值。

Description 描述

有了这个变量，你可以获得或设置视野取景的旋转角度。默认是零，但是可以设置为-359到359的任意角度。这只是设置视野在房间里取景的角度，不会影响游戏展示的角度。它经常用来创建旋转的小地图，或者震动效果（下面有举例说明）。

注意: 这个变量不支持HTML5目标模块和Windows 8 (JavaScript)模块。

With this variable, you can get (or set) the angle at which the view is to be drawn. The default angle is 0?but it can be set to any angle between -359?and 359? This will not affect how the game is played as it is only for setting the angle at which the view is drawn. Often it is used to create a rotating minimap, or to "shake" the view for example.

NOTE: This variable will not work on the HTML5 target module, nor the Windows 8 (JavaScript) module.




Example 举例:

if shake
   {
   view_angle[0] = -10 + random(20);
   }
else
   {
   view_angle[0] = 0;
   }

上述代码将检测 "shake" 变量的值，如果返回真，它将view[0]的角度每步更改为-10到10之间的随机值，直至 "shake" 变量返回为假，然后view[0]的角度又回到零。

the above code will check the variable "shake" and if it returns true, it will then change the view[0] angle to a random value between -10 and 10 every step until the variable returns false, at which time it will return the view[0] angle to 0?
