view_surface_id

为视野指定一个表面。Assign a surface to a view.

Syntax 用法:

view_surface_id[0...7]


Returns 返回: Real 实数


Description 描述

有了这个变量，你可以获取或设置一个视野的表面。 当用到表面时，常常要求将整个屏幕内容绘制到表面上，所以你将用这个函数为表面指定一个视野。意味着所有选定视野显示的内容将绘制到指定的表面上，你也可以读取这个变量以获得表面的索引，这个索引指向选定的视野，如果没有指定视野则返回-1。更多表面的信息请看 Reference - Surfaces这一节内容。

With this variable you can get (or set) the surface of a view. When working with surfaces, it is often required that you draw the whole visible region of the screen to the surface, and so you would assign it to a view using this function. This means that everything that is shown in the chosen view will also be drawn to the assigned surface. You can also read this variable to get the index of the surface that has been assigned to the chosen view or it will return -1 if no surface has been assigned. For more information on surfaces see the section Reference - Surfaces.


Example 例子:

surf = surface_create(room_width, room_height);
view_surface_id[0] = surf;

The above code creates a surface and assigns it's index to the variable "surf". The code then assigns this surface to view[0].

上述代码将创建一个表面，给它的索引指定变量“surf”，然后指定这个表面给view[0]。
