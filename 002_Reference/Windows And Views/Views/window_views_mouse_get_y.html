window_views_mouse_get_y

相对于窗口内给定的所有激活的视野，返回鼠标位置的Y坐标。RReturns the y-coordinate of the mouse with respect to the views.

Syntax 用法:

window_views_mouse_get_y();


Returns 返回: Real 实数


Description

这个函数将返回相对于所有激活视野的鼠标位置的Y坐标，和mouse_y返回相同的坐标值。

This function returns the y-coordinate of the mouse with respect to all the active views and returbns the same value mouse_y.


Example :

myy = window_views_mouse_get_y();

上述代码将设置myy变量为窗口内鼠标位置y坐标，这个鼠标位置坐标相对于所有激活的视野。

This would set myy to the y coordinate of the mouse in the window relative to all active views.
