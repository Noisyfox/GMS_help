Windows And Views 窗口与视野

本章节涉及函数是为了操作视角和游戏窗口。

“窗口”是 GameMaker:Studio 显示你的游戏的地方，并根据目标设备，可以用它做不同的事情，像将其设置为全屏或不（例如）。然而，官方的就是在该窗口中看到什么，因此有另一系列的函数专用于他们。以下两部分涉及显示你的游戏的这两个方面：

  1. The Game Window 游戏窗口
  2. Views 视野

然而，有时它不足以获得游戏窗口和只是视角的信息，你真的需要知道显示器本身的事情。下面的函数给你的关于显示的大小和方向以及在它的实际的鼠标坐标内的详细信息：

  1. display_reset
  2. display_get_height
  3. display_get_width
  4. display_get_orientation
  5. display_get_dpi_x
  6. display_get_dpi_y
  7. display_set_gui_size
  8. display_get_gui_width
  9. display_get_gui_height
  10. display_mouse_get_x
  11. display_mouse_get_y
  12. display_mouse_set

也有一对的特殊函数，可用于保存显示，或它的一部分为图像文件：

  1. screen_save
  2. screen_save_part
