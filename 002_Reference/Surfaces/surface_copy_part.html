surface_copy_part

复制源表面图层的一部分到目标表面图层。Copies part of a source surface into a destination surface. (Without any form of blending.)

Syntax  用法:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);


Argument Description
destination 目标表面的索引。The index of the surface to copy the other surface to.
x 复制到的x坐标。The x position to copy to.
y 复制到的y坐标。The y position to copy to.
source 源表面索引。The surface to be copied.
xs 源表面被复制的部分的x坐标。The x position in the source surface to copy from.
ys 源表面被复制的部分的y坐标。The y position in the source surface to copy from.
ws 源表面被复制的部分的宽度。The width of the area in the source surface to copy from.
hs 源表面被复制的部分的高度。The height of the area in the source surface to copy from.


Returns 返回:N/A

 

Description 描述

这个函数简单的从一个表面获取图像，复制到另外一个表面的特定位置（ 如(0,0)，是目标表面图层的左上角）。你可以选定源表面的一个特定坐标和大小的区域进行复制。请注意，坐标是基于表面的起始点，而不是表面所在的房间（查看 Surfaces主页获得更多的信息）。如果目标表面已经有内容了，将会被拷贝版本重写，而这个函数不会通过任何方式改变源表面的内容。

注意 :当表面工作的时候，因为储存在显存里，可能会随时停止工作并退出。在直接引用之前，你应该一直使用 surface_exists   检测这个表面是否会退出，更多信息请查阅 Surfaces。 

This function simply takes the image from one surface and copies it onto another one at the specified local position within that surface (where the (0,0) position is the top left corner of the destination surface). You can specify a local x and y position to copy from as well as the width and height of the section. Please note that these are coordinartes based on the surface sizeand not on the position at which the surface is being drawn in the room (see the main page on Surfacesfor further information). If the destination surface already has information this will be over-written by the copy, and the function does notchange the source surface in any way.

NOTE :When working with surfaces there is the possibility that they can cease to exist at any time due to them being stored in vRam. You should ALWAYScheck that a surface exists using surface_exists before referencing them directly. For futher information see Surfaces.
 

 

Example 例子:

if view_current = 0
   {
   view_copy(surf, 0, 0, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse-y, temp_surf);
   }
else
   {
   draw_surface(surf, 0, 0);
   }

上述代码将检测当前被绘制的视野是否为view[0]，如果是，则复制变量为"temp_surf"的表面到变量为 "surf"的表面。被复制的区域相当于一个矩形，截取自表面内相对鼠标的位置，它将在view[1]里绘制。如果检测的不是 view[0]，将直接在屏幕上绘制 "surf"表面。

The above code will check the current view being drawn and if it is view[0] it copies the surface indexed in the variable "temp_surf" onto the surface indexed in the variable "surf". the area copied corresponds to a rectangle formed by the relative position of the mouse within the surface as it would be drawn in view[1]. If the current view is anything other than view[0] the surface "surf" is drawn to the screen.

注：这个例子用了view_copy的函数，并没有用到surface_copy_part，所以会不会执行成功呢。
