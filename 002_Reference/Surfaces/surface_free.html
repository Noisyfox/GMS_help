surface_free

破坏表面图层，并从内存中释放。Destroys the surface, freeing it from memory.

Syntax 用法:

surface_free(index);

 

Argument 参数 Description 描述
index 被释放的表面图层的索引。The index of the surface to be freed.

 

Returns 返回: N/A

 

Description 描述

当你用到表面图层时，不论是否结束使用，你应该一直使用这个函数。表面图层会占据内存空间，通常需要在房间（room）结束工作时清除它们，可以在任何时间执行这个过程，取决于你使用的方式。如果不这样做，可能会导致内存泄漏，最终放慢游戏甚至游戏崩溃。

注意 : 当正在使用表面图层时，由于存储在显存里，它可能会随时停止工作。在直接引用它们之前，你应该使用 surface_exists 这个函数保持检测是否表面图层还继续存在。更多信息请查看Surfaces这篇的介绍。

When you are working with surfaces, you should always use this function whenever you are finished using them. Surfaces take up space in memory and so need to be removed, normally at the end of a room, but it can be done at any time depending on the use you put them to. Failure to do so can cause memory leaks which will eventually slow down and crash your game.

NOTE : When working with surfaces there is the possibility that they can cease to exist at any time due to them being stored in vRam. You should ALWAYS check that a surface exists using surface_exists before referencing them directly. For futher information see Surfaces.
 

Example 例子:

if keyboard_check_pressed(vk_escape)
   {
   surface_free(surf);
   room_goto(rm_Menu);
   }

上述代码将检测一个键是否按下，如果检测到了，则对索引为变量surf的表面图层所预留的那部分内存进行释放，然后切换一个房间。

The above code checks for a key press and if it detects one it frees the memory reserved for the surface indexed in the variable "surf" and then changes room.
