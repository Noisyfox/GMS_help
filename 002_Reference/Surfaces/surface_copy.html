surface_copy

复制源表面图层到目标表面。Copies a source surface into a destination surface.

Syntax 用法:

surface_copy(destination, x, y, source);

 

Argument Description
destination 需要复制到其它表面图层的表面索引。The index of the surface to copy the other surface to.
x 复制到的x坐标。The x position to copy to.
y 复制到的y坐标。The y position to copy to.
source 被复制的表面图层。The surface to be copied.

 

Returns 返回:N/A

 

Description 描述

这个函数简单的让图像从一个表面图层复制到另外一个表面特殊的位置（(0,0)表示在表面图层的左上角）。如果目标表面已经有内容了，拷贝版本将会重写表面，这个函数不会用任何方式改变源表面图层。

注意 :当表面图层工作的时候，由于储存在显存里，是有可能随时停止并退出的。在直接引用他们之年， 你需要一直使用 surface_exists 检测表面是否退出。更多信息请查看 Surfaces。

This function simply takes the image from one surface and copies it onto another one at the specified local position within that surface (where the (0,0) position is the top left corner of the destination surface). If the destination surface already has information this will be over-written by the copy, and the function does notchange the source surface in any way.

NOTE :When working with surfaces there is the possibility that they can cease to exist at any time due to them being stored in vRam. You should ALWAYScheck that a surface exists using surface_exists before referencing them directly. For futher information see Surfaces.

 

Example 例子:

if view_current = 0
   {
   view_copy(surf, 0, 0, temp_surf);
   }
else
   {
   draw_surface(surf, 0, 0);
   }

上述代码将检测当前绘制的视野，如果视野是 view[0]，则复制变量为 "temp_surf" 的表面到变量为 "surf"的表面。如果当前视野不是view[0]，屏幕上则绘制 "surf"这个表面。

The above code will check the current view being drawn and if it is view[0] it copies the surface indexed in the variable "temp_surf" onto the surface indexed in the variable "surf". If the current view is anything other than view[0] the surface "surf" is drawn to the screen.
