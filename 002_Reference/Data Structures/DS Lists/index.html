DS Lists 列表

ds_list 存储的值的集合在一个特定的顺序。

ds_list是一种数据结构，按添加的顺序存储信息（很像一个一维数组 1D array）。DS列表是非常灵活的数据结构，允许你添加值在尾部或插入他们在列表中间，以及给你随机排序（初始化函数值）或对它们进行排序以递增或降序排列。

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。

存在以下函数处理DS 列表：

  1. ds_list_create
  2. ds_list_destroy
  3. ds_list_clear
  4. ds_list_empty
  5. ds_list_size
  6. ds_list_add
  7. ds_list_delete
  8. ds_list_find_index
  9. ds_list_find_value
  10. ds_list_insert
  11. ds_list_replace
  12. ds_list_shuffle
  13. ds_list_sort
  14. ds_list_copy
  15. ds_list_read
  16. ds_list_write
