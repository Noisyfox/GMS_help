DS Stacks 堆栈

ds_stack 是一个后进先出数据结构。

堆栈的数据结构是所谓的后进先出（LIFO）的结构。您可以推值到堆栈，并通过从堆栈中弹出来再次删除它们，最近被压入堆栈的值总是第一个被弹出 （想想一叠硬币，其中每个你添加的硬币，首先你得到其余下的硬币之前要再次被删除最上面的）。堆栈经常被用于中断处理，或递归函数时，甚至为您的游戏构建 一个基本的AI。

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。

存在以下函数处理DS堆栈。

  1. ds_stack_create
  2. ds_stack_destroy
  3. ds_stack_clear
  4. ds_stack_empty
  5. ds_stack_size
  6. ds_stack_copy
  7. ds_stack_top
  8. ds_stack_pop
  9. ds_stack_push
  10. ds_stack_read
  11. ds_stack_write
