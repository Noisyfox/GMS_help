DS Grids 网格

本章节解释各类关于使用和操作 ds_grids 的函数。

ds_grid 是二维数组 two-dimensional array的一个基本类型，您可以通过设置他的证书宽度和高度定义网格，然后就设定网格的“细胞”总数（乘以的宽度和高度），这些细胞被用来存储不同的数据值（实数或字符串）。
这种类型的结构允许设置和检索在网格中的细胞的值，通过给予 的x和y的索引（以0开始的x和y轴和上升- 1的宽度和高度的值-1）。但你也可以设置特定的网格区域中的值，添加单个值，和检索的总和，最大值，最小值，平均值在一个给定的区域，使得这个数据结构 提供给您更多样化。

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。
