iap_acquire

This requests acquisition of the given product.

Syntax :

iap_acquire(purchase_id);


ArgumentDescription
purchase_id This can either be the Apple Store/Google Play id string, an URL id, or an index into the available purchases ds_list.


Returns : N/A


Description

This function is the one that you use to aquire a purchase through the app store for the device running your game. You can specify the Apple Store/Google Play id string, or you can pass through the actual identifier name(eg: ".extralevels"), or the position from within the available purchase list of the product to be bought. If you use the list approach, the number you input will correspond to the position of the product within the ds_list as defined by the function iap_activate, and GameMaker:Studio will automatically get the product id for you and fullfill the purchase order.


Example :

if mouse_check_button_pressed(mb_left)
   {
   iap_acquire(".extralevels")
   }

The above code will send a purchase request when the mouse button is pressed.
