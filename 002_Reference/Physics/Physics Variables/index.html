Physics Variables 物理变量

对象有某些变量与他们相关，会自动使用在物理世界。

与“传统”的碰撞和运动系统一样，物理系统也有一系列的实例变量在创建时分配给每一个对象的实例。其中的某些属性可以由用户在GML定义，但很多是只读的，并且更多的只适用于碰撞事件。

变量

存在下列变量在所有的物理功能的物件中，他们可以通过实例和脚本的代码读取和写入（即：改变）：

  1. phy_active
  2. phy_angular_velocity
  3. phy_linear_velocity_x
  4. phy_linear_velocity_y
  5. phy_speed_x
  6. phy_speed_y
  7. phy_position_x
  8. phy_position_y
  9. phy_rotation
  10. phy_fixed_rotation
  11. phy_bullet

只读变量

某些变量是只读变量，这意味着通过实例代码或脚本所包含的代码不能改变（但它可以通过GameMaker:Studio内部改变，这就是为什么这些是变量，而不是常量）。下表显示的只读的变量存在于所有物理系统激活的对象中：

  1. phy_com_x
  2. phy_com_y
  3. phy_dynamic
  4. phy_kinematic
  5. phy_inertia
  6. phy_mass
  7. phy_sleeping

碰撞唯一变量

最后，还有一些临时变量中存在的物理激活的对象中，并且仅在碰撞事件被创建。这些变量和上面列出的一样是只读的：

  1. phy_collision_points
  2. phy_collision_x
  3. phy_collision_y
  4. phy_col_normal_x
  5. phy_col_normal_y
