Fixtures 固定

GameMaker:Studio 物理系统有许多函数可于定义被实例支持的固定。

在 GameMaker:Studio 的实例有物理性质之前，它必须有一个固定分配给它。固定限制了形状或形式到实例，并且增加了材料属性，如密度，摩擦和还原等。固定可以是盒，圆，多边形或任何其他简单的形状， GameMaker:Studio 如你所希望那样提供了函数，使您可以逐点定义您自己的形状，然后，一旦你有你的形状，你可以继续并定义其物理性能。这些属性是在物理世界里支配将与分配给它的固定的实例与其他实例反应的方法，它们在下面列出：

  • 密度: 材料的密度被定义为每单位体积质量。这是从本质上讲，测量是多么紧紧挤在一起的。如果您希望您的固定是静态 的（即：不动产）则密度应设置为0，本质上使其无限致密。运动的对象以恒定速度移动，并无实际的物理碰撞影响的话，也被定义使用的密度为0。

  • 摩擦 : 摩擦是抵制固体表面的相对运动，抵抗流体层和物质要素相互滑动的力。

  • 线性阻尼 : 这是广义的力量，使移动物体失去动力。在“现实世界”的就永远不发生。例如，A纸飞机飞很短的一段时间，但最终将空气摩擦作用于它，导致它慢下来。在物理世界，我们创造这样的事情是困难的，所以每个固定给出了自己的非线性阻尼来模拟这种类型的“广义摩擦”。

  • 角阻尼 : 如上文所述，这是一个广义的力量，适用对象永久停止旋转。

  • 还原: 在物理学上，还原的定义为“一个对象或系统在弹性形变后返回到其原来的状态”，但作为GameMaker:Studio 的固定是真正的刚性机构，并不能变形，认为这基本上是固定是如何“弹性”的。

所以你可以看到，只需几个简单的功能，GameMaker:Studio 可以帮助你创建复杂的系统和互动，这将给你的游戏了前所未有的真实感。


Note : 你可以定义一个固定，并结合多个实例（想想一个台球桌，所有的球具有相同的物理性质，但每一个都是一个不显眼的实体）


创造固定

以下函数是创建，绑定，设置碰撞，然后删除固定：

  1. physics_fixture_create
  2. physics_fixture_bind
  3. physics_fixture_set_collision_group
  4. physics_fixture_delete

定义固定形状

固定必须在给定的形状，否则在物理世界中它不会被检测到的，这种形状可以定义由以下函数设定：

  1. physics_fixture_set_box_shape
  2. physics_fixture_set_circle_shape
  3. physics_fixture_set_polygon_shape
  4. physics_fixture_add_point

固定属性

为了使你的物理实例在它周围的世界作出适当的反应，你所使用的固定必须有特定的属性，这将使夹具反弹，摩擦，质量等.. 使用下面的函数来设置这些固定的属性：

  1. physics_fixture_set_density
  2. physics_fixture_set_friction
  3. physics_fixture_set_linear_damping
  4. physics_fixture_set_angular_damping
  5. physics_fixture_set_restitution
  6. physics_fixture_set_sensor
  7. physics_fixture_set_kinematic
  8. physics_fixture_set_awake
