Obsolete File Functions 淘汰文件函数

这些函数是之前GM的一般函数。

Variable Function List :

FunctionDescription
file_open_write开 启指定文件名称的文件进行写入。                                                               
file_open_read开启指定文件名称的文件进行读取。
file_open_append开启一个文件准备附加信息
file_write_string写入字串到指定文件id的文件。
file_write_real写入实数到指定文件id的文件。
file_writeln写入换行字符到指定文件id的文件。
file_read_string从指定文件id的文件读取一个字串并返回这个字串。
file_read_real从指定文件id的文件读取一个实数并返回这个实数。
file_readln跳过文件中一行的其余部分，从下一行的行头开始读取。
file_eof返回是否到达文件的终结。
file_eoln返回本行是否到达文件的终结。
file_close关闭打开的文件

为什么淘汰这些

因为已经被 file_text_* 和 the file_bin_* 取代。
