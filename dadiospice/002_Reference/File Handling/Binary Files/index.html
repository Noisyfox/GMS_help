Binary Files 二进制文件

章节是说明关于二进制文件的函数。

 在极少数情况下，您可能需要从二进制文件中读取数据。

NOTE :  这些函数不能工作在HTML5目标模式下。

存在如下低级例程：

  1. file_bin_open
  2. file_bin_rewrite
  3. file_bin_close
  4. file_bin_size
  5. file_bin_position
  6. file_bin_seek
  7. file_bin_write_byte
  8. file_bin_read_byte

NOTE : 这些函数被限制在标准版 Standard Version 的 GameMaker:Studio 及其以上。
