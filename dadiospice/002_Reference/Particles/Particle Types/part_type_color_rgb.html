part_type_color_rgb

Set the particle color from a red, green and blue range.

Syntax :

part_type_color_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>


ArgumentDescription
ind The index of the particle type to change.
rmin The minimum the final color's red component can be.
rmax The maximum the final color's red component can be.
gmin The minimum the final color's green component can be.
gmax The maximum the final color's green component can be.
bmin The minimum the final color's blue component can be.
bmax The maximum the final color's blue component can be.


Returns : N/A


Description

With this function you can set the mix of red, green and blue colors for all particles created of the given type. You supply a minimum value and a maximum value for each of the three components and the particles created will have a random color based on the given range of parameters. All values must be between 0 and 255.


Example :

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

The above code sets each particle emitted of the particle type indexed in the global variable "Blood_Part" to be only different shades of red.
