Strings 字符串

以下函数是为处理字符串存在的。

在你制作游戏的某些时候，你需要使用文本。在游戏中的文本需要使用字符串函数（字符串只是表述文本行的另外一种方式）， GameMaker:Studio 拥有一套完整的设置函数，允许您以多种方式操作字符串，包括插入一个字符串到另一个字符串中，复制字符串，分析字符串包含的数字或字母。

下面列出的函数是处理字符串：

  1. ansi_char
  2. chr
  3. ord
  4. real
  5. string
  6. string_byte_at
  7. string_byte_length
  8. string_set_byte_at
  9. string_char_at
  10. string_copy
  11. string_count
  12. string_delete
  13. string_digits
  14. string_format
  15. string_insert
  16. string_length
  17. string_letters
  18. string_lettersdigits
  19. string_lower
  20. string_pos
  21. string_repeat
  22. string_replace
  23. string_replace_all
  24. string_upper
  25. string_height
  26. string_height_ext
  27. string_width
  28. string_width_ext

除了​​这些函数，具体涉及到字符串中，在Windows目标下还允许您访问剪贴板获取和设置文本信息：

  1. clipboard_has_text
  2. clipboard_get_text
  3. clipboard_set_text
