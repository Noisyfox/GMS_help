os_lock_orientation

this function can be used to lock the orientacion to that chosen in the global game settings.

Syntax :

os_lock_orientation(flag)


ArgumentDescription
flag Set to true or false to enable or disable orientacion locking.


Returns : N/A


Description

With this function you can set your device to "lock" to the chosen orientacion (as set by the Global Game Settings tab for the target device). This mainly applies to iOS devices where you can have a "left" and "right" orientation, meaning that a tilt game may actually be reversed if the device is tilted too far, in which case you would want this flagged as true to prevent this flipping of the game.


Example :

if os_type == os_android || os_type == os_ios
   {
   os_lock_orientation(true);
   }

The above code checks the OS type and if it is either Android or an iOS then the orientacion locking is flagged as true.
