本章节包括 Game Maker 语言的全部信息

GameMaker:Studio 包含内置程序语言，一般称为GML，它将比使用拖放的动作给与您更多灵活性和可操作性。在这个章节中我们将讲述GML的使用、概述全部函数和控制游戏的各种变量。
