Instance Properties 实例属性

本章节介绍了实例中的所有内置变量。

当一个实例被创建时，GameMaker:Studio中生成了大量的预定义变量，以帮助您设置和更改某些共同的特性，，如sprite_index，或深度。下面你可以找到所有这些内置的变量的列表。

一般变量

这些变量处理一般的实例属性:

  1. id
  2. solid
  3. visible
  4. persistent
  5. depth
  6. alarm
  7. object_index

精灵变量

这些变量处理一般的实例属性:

  1. sprite_index
  2. sprite_width
  3. sprite_height
  4. sprite_xoffset
  5. sprite_yoffset
  6. image_alpha
  7. image_angle
  8. image_blend
  9. image_index
  10. image_number
  11. image_speed
  12. image_xscale
  13. image_yscale

遮罩函数

这部分函数是处理碰撞遮罩:

  1. mask_index
  2. bbox_bottom
  3. bbox_left
  4. bbox_right
  5. bbox_top

额外变量

如下是多个专业的变量，使用不像上面的变量那么多，因为这些的变量涉及到具体的GameMaker:Studio 某个方面，并非所有的游戏都会利用。以下各节关于这些变量：

  1. Movement Variables 移动变量
  2. Path Variables 路径变量
  3. Timeline Variables 时间线变量
  4. Physics Variables 物理变量
