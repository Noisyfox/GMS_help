sound_delete

Deletes the indicated sound to free it from game memory.

Syntax :

sound_delete(index);


ArgumentDescription
index The index of the sound to delete.


Returns : N/A


Description

This function will delete the indexed sound from your game.

NOTE : This is a permanent removal that lasts as long as the game is running, meaning that if the deleted sound is an included game asset, even restrating the game with game_restart will not restore the sound. For that the game must be exited and re-loaded again.

NOTE : This functionality is limited to the Standard Version of GameMaker:Studio and above.


Example :

if sound_exists(global.Music)
   {
   sound_delete(global.Music);
   }

The above code checks to see if the global variable "Music" indexes a sound and if it does it then deletes that sound from the game.
