background_showcolor

This variable determines if the current room's background color should be drawn.

Syntax:

background_showcolor


Returns : Boolean


Description

When in a room, it can have no background image selected for any of it's background_index, in which case you should set a background color with the background_color variable and make sure that it is visible by setting this variable is set to true. Having a room with no background image and no background color will result in graphical artefacts being drawn on the screen and is not recommended.


Example :

if background_index[0] == -1
   {
   background_color = c_black;
   background_showcolor = true;
   }

The above code checks to see if background0 has an image assigned to it and if not it sets the background color to black and flags it to be shown.
