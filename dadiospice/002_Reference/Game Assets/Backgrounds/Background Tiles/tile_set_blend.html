tile_set_blend

Sets a tile's blending.

Syntax :

tile_set_blend(index, color);


ArgumentDescription
index The index of the tile to set.
color The color to blend the tile with.


Returns : N/A


Description

With this function you can set a color to be blended with the tile. The default value for the blend color is -1 (which represents the constant c_white) but any color value can be used. This is not permanent and if the player leaves the room and returns to it again, the tile will have it's original blend color (if it was placed in the room editor and the room is not persistent).


Example :

var tile;
tile = tile_layer_find(-1000, mouse_x, mouse_y);
if tile_get_blend(tile) != -1
   {
   tile_set_blend(tile, c-white);
   }

The above code will first get the index of a tile at the mouse position with a depth of -1000 and then use that value to check whether the tile blend is anything other than the default -1. If it is it sets the tile blend to white (no blending).
