font_get_texture

Returns the texture id for the given font.

Syntax :

font_get_texture(font);


ArgumentDescription
font The index of the font to use.


Returns : Real


Description

This function returns a special id for the font texture. This value can then be used in other draw functions, particularly in general 3D and some of the 2D primitive functions, as well as the Shader functions.


Example :

tex = font_get_texture(fnt_Main);

The above code will get the texture id for the font indexed as "fnt_Main".
