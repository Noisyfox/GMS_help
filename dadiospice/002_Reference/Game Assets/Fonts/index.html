Fonts 字体

本章节涉及可用于字体的函数。

有许多的函数，可用于获取信息字体资产，同样也有一组更强大的函数，可用于从您的游戏中改变，增加或删除字体在运行时。

字体信息

如下函数可被用于获得关于字体资源的信息：

  1. font_get_name
  2. font_get_fontname
  3. font_get_first
  4. font_get_last
  5. font_get_italic
  6. font_get_bold
  7. font_get_size

字体操作

如下函数可被用于在游戏时改变和操作字体：

  1. font_set_cache_size
  2. font_exists
  3. font_add
  4. font_add_sprite
  5. font_add_sprite_ext
  6. font_replace
  7. font_replace_sprite
  8. font_delete
