d3d_transform_set_rotation_y

Sets the transformation to a rotation around the y-axis with the indicated amount.

Syntax :

d3d_transform_set_rotation_y(angle)


ArgumentDescription
angle The angle to rotate around the y-axis.


Returns : N/A


Description

This function will rotate the object around the y-axis by the specified amount (in degrees).


Example :

d3d_transform_set_identity();
d3d_transform_set_rotation_y(90);
d3d_draw_block(-50, -50, -50, 50, 50, 50, 100, 100, 100);
d3d_transform_set_identity();

The above code tells GameMaker:Studio to rotate the block 90� along the y-axis before drawing it.
