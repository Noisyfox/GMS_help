Drawing 绘制

本章节包括GameMaker:Studio全部各种关于绘制的函数

当一个对象分配给它一个精灵，它会绘制这个精灵，而不需要添加任何绘制事件。但也有许多场合，你将需要添加其他东西进绘制事件的对象，本节中的所有功能，可用于不同的事情，你可以借鉴，从简单的圆形和矩形，到精灵与文本，并通过二维图元，曲面和3D：

  1. Color And Blending 颜色与混合
  2. Drawing Basic Forms 绘制基本形式
  3. Drawing Sprites And Backgrounds 绘制精灵与背景
  4. Drawing Text 绘制文本
  5. Drawing Primitives 绘制图元
  6. Drawing 3D 绘制3D
  7. Drawing Surfaces 绘制表面

也有一些特殊的函数关系着GameMaker:Studio绘制纹理使用的方式。请了解一个精灵实际上是一个四点纹理（四点图元做出两个三角形），而且这样方式也会影响到他们是如何得出以下函数：

  1. texture_get_width
  2. texture_get_height
  3. texture_set_interpolation
  4. texture_set_blending
  5. texture_set_repeat

最后，还有一个额外的绘图功能，清除装置记忆体中的任何纹理页面：

  1. draw_texture_flush
