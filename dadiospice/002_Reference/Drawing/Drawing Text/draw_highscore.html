draw_highscore

Draws the highscore table in the room in the indicated box using the current font.

Syntax :

draw_highscore( x1, y1, x2, y2 );


ArgumentDescription
x1 The x coordinate of the left of the highscore rectangle.
y1 The y coordinate of the top of the highscore rectangle.
x2 The x coordinate of the right of the highscore rectangle.
y2 The y coordinate of the bottom of the highscore rectangle.


Returns : N/A


Description

This simple function will draw the current list of internally stored high scores using the currently set font, color and alpha values within the specified rectangle. You provide the coordinates for the upper left corner and lower right corner of the rectangular are to draw the text, and GameMaker:Studio will take care of the rest, with spacing and position being done automatically.


Example :

draw_highscore(100, 100, room_width - 100, room_height -100);

This would draw the highscore table in a rectangle in the middle of the room with a 100px border.
