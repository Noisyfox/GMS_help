merge_color

Merges two colors by a given amount.

Syntax:

merge_color(col1, col2, amount);


ArgumentDescription
col1 The first color to merge
col2 The second color to merge
amount How much of each color should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colors equally


Returns : value


Description

With this function you can take two colors and then merge them together to make a new color. The amount of each of the component colors can be defined by changing the "amount" argument, where a value of 0 will return the first color (col1), a value of 1 will return the second color (col2) and a value inbetween will return the corresponding mix. For example, a value of 0.5 will mix the two colors equally. The following image illustrates how this works by merging the colors red and blue together:


Example :

col = merge_color(c_lime, c_orange, 0.5);

The above code uses the function to create a color by merging lime and orange 50/50 and then stores its value in the variable "col" for later use.
