color_get_value

Returns the value of the color given.

Syntax:

color_get_value(col);


ArgumentDescription
col The color to check


Returns : Real


Description

This function will return the value (luminosity) of the given color. This is the amount of the "light" that is mixed into the final color and is part of the hue, saturation and value method for defining a color. The following image illustrates how this value corresponds to the HSV scale of color:


Example :

col = make_color_hsv(random(255), 255, color_get_val(c_teal));

The above code gets the value used to make the color constant "c_teal" and then uses that value to set a random color to have the same luminosity, storing the resulting color in the variable "col".
