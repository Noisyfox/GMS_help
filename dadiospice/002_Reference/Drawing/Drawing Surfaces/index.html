Drawing Surfaces 绘制表面

本章节关于绘制表面。

表面是一个令人难以置信的强大的图形工具，GameMaker:Studio为您提供创建特效和一大堆其他的事情。他们使用来说是相对简单的，因为它们基本上是一个画布，代替显示屏幕绘制，这个画布可以操纵和改变之前此页上的函数实现的绘制。欲了解更多如何创建表面，绘制他们，并与其使用的缺点的信息，请参阅表面 Surfaces。

NOTE : 当表面工作时，由于他们被存储在vRAM，有可能他们可以在任何时候停止存在。你应该总是使用 surface_exists 函数检查一个表面的存在，在直接引用他们之前。如需进一步信息，请参阅表面 Surfaces.

存在以下函数关于表面 ：

  1. draw_surface
  2. draw_surface_ext
  3. draw_surface_part
  4. draw_surface_part_ext
  5. draw_surface_stretched
  6. draw_surface_stretched_ext
  7. draw_surface_tiled
  8. draw_surface_tiled_ext
  9. draw_surface_general
