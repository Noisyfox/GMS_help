draw_surface_stretched_ext

Draws the surface stretched to an indicated region with color and alpha blending.

Syntax :

draw_surface_stretched_ext( index, x, y, w, h, color, alpha );


ArgumentDescription
index The index of the surface to draw.
x The x position of where to draw the surface.
y The y position of where to draw the surface.
w The width at which to draw the surface.
h The height at which to draw the surface.
color The color with which to color the surface.
alpha The alpha with which to blend the surface.


Returns : N/A


Description

This function does exactly the same as the draw_surface_stretched function with the added ability to set the color blending and alpha value for the surface when it is drawn (similar to the function draw_surface_ext).

NOTE : When working with surfaces there is the possibility that they can cease to exist at any time due to them being stored in vRam. You should ALWAYS check that a surface exists using surface_exists before referencing them directly. For futher information see Surfaces.


Example :

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

This will draw the given surface with its left corner at the instances x/y position and it will be stretched to occupy an area of 200x200 pixels with no blending, but partial transparency.
