Drawing Basic Forms 绘制基本形式

本章节展示了关于绘制基本形状和形式的函数

GameMaker:Studio 有一整个系列函数，可绘制不同的形式和形状。应该只使用这些函数在实例的绘制事件中（你可以在其它地方使用，但你不会看到任何东西！） 并创建纯粹的图形效果，这意味着它们不会产生碰撞的事件，只能做的是通过精灵或碰撞遮罩分配给一个实例。所有这些函数都受到最近绘制颜色，透明度和混合模 式的影响（见颜色和混合 Color And Blending）。

绘制形状

存在以下的绘图函数，可以用来绘制基本形状：

  1. draw_arrow
  2. draw_circle
  3. draw_circle_color
  4. draw_ellipse
  5. draw_ellipse_color
  6. draw_line
  7. draw_line_color
  8. draw_line_width
  9. draw_line_width_color
  10. draw_point
  11. draw_point_color
  12. draw_rectangle
  13. draw_rectangle_color
  14. draw_roundrect
  15. draw_roundrect_color
  16. draw_triangle
  17. draw_triangle_color

高级绘图函数

在GameMaker:Studio 中为帮助更好地控制你在屏幕上的绘制，还包括以下函数：

  1. draw_set_circle_precision
  2. draw_button
  3. draw_healthbar
  4. draw_path
