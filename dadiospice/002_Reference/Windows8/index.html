Windows 8

这些函数是专门提供为windows 8 平台的。

Windows 8的模块有一些是专门设计用于针对它的游戏功能。这些函数大多数是Windows 8瓷砖系统的，以及允许您访问和共享你的游戏文件。如下函数已经被分成了以下部分使您更容易找到合适您的需求的内容。请注意，这些函数是Windows 8的桌面和表面平台，不是Windows 8电话。对于Windows 8的电话，请参阅以下部分：

  1. Windows 8 Phone

NOTE : 虽然Windows 8的目标与HTML5相似，它不会允许同一应用程序的多个实例运行在同一时间，如果该文件被运行时，它不能覆盖它。这意味着你将不得不关闭该应用程序之前保存/测试一个新的版本。

动态瓷砖

随着Windows 8操作系统中你的游戏（或应用程序）与它自己的动态瓷砖联系在一起。一个动态瓷砖是你游戏的实时更新的一个动态的链接，这意味着，你可以“推”的图像和文字到它上面，在用户在您的游戏的特定的时间的时候。以下功能是推荐的方法，用于创建动态瓷砖的通知：

  1. win8_livetile_notification_begin
  2. win8_livetile_notification_secondary_begin
  3. win8_livetile_notification_expiry
  4. win8_livetile_notification_tag
  5. win8_livetile_notification_text_add
  6. win8_livetile_notification_image_add
  7. win8_livetile_notification_end

然而，还存在以下旧式功能（只应在绝对必要时使用）：

  1. win8_livetile_tile_notification

除了通知推到主要瓷砖也有一些进一步的事情是可以做的，比如添加一个徽章或清除瓷砖，使用以下功能：

  1. win8_livetile_tile_clear
  2. win8_livetile_badge_notification
  3. win8_livetile_badge_clear
  4. win8_livetile_queue_enable

您也有权访问辅助动态瓷砖。这是一个辅助动态瓷砖类型，可以被固定到初始页面，用于访问你的游戏的特定部分，例如保存的游戏。设置它是相当棘手的，就像在辅助瓷砖在你的游戏中被访问时，你需要提供的特定的命令行参数将被解析那样工作（使用 parameter_string 函数）从。以下是所有辅助瓷砖功能列表：

NOTE: 辅助瓷砖将包括所有用于创建标准宽瓷砖的强制属性。

  1. win8_secondarytile_pin
  2. win8_secondarytile_badge_notification
  3. win8_secondarytile_delete

你可以找到一个动态瓷砖的功能在这里的一些可用的模板字符串列表：

  1. Windows 8 Live Tile Templates

应用工具条命令

应用程序的条是一个特别棒的按钮，可以通过右键点击你的游戏弹出。通过这些按钮可以定义以下的功能和脚本在GameMaker:Studio中创建暂停菜单，选项等.. ：

  1. win8_appbar_enable
  2. win8_appbar_add_element
  3. win8_appbar_remove_element

设定charm

当在你的游戏中，如果你将鼠标移动到的屏幕的右上角（或右下角），一个侧栏的幻灯片，显示了不同的“charm”（图标）。这些函数使您能够添加自定义选项的“设置”（Settings）的charm，无论是通过内部脚本或外部HTML文件：

  1. win8_settingscharm_add_entry
  2. win8_settingscharm_add_html_entry
  3. win8_settingscharm_remove_entry

分享

这些函数实际上是一个先进的“剪切”和“粘贴”选项为Windows 8，并允许你发送的信息以文本，图像，文件，甚至一个URL：

  1. win8_share_image
  2. win8_share_screenshot
  3. win8_share_file
  4. win8_share_url
  5. win8_share_text

搜索

Windows 8可以使你的游戏通过任何其他窗口的搜索系统。通过这些函数控制：

  1. win8_search_enable
  2. win8_search_add_suggestions
  3. win8_search_disable
