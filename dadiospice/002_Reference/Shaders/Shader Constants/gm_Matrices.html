gm_Matrices

Returns the index to one of the available transform matrices.

Syntax :

gm_Matrices[matrix constant];


Returns: Matrix index


Description

This array constant returns a transform matrix and is one of the available pre-defined uniforms that GameMaker:studio creates for you to use within the shader code editor. The array index is chosen from one of the following index constants:

  1. MATRIX_VIEW
  2. MATRIX_PROJECTION
  3. MATRIX_WORLD
  4. MATRIX_WORLD_VIEW
  5. MATRIX_WORLD_VIEW_PROJECTION


Example :

vec4 pos = gm_Matrices[MATRIX_VIEW] * objectspacepos;

The above example is a snippet of GLSL code using the gm_Matrices array constant.
