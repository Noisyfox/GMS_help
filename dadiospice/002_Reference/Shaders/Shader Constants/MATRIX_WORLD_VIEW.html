MATRIX_WORLD_VIEW

The current world and view matrices multiplied together as a uniform for use in shaders.

Syntax :

MATRIX_WORLD_VIEW;


Returns: Matrix array index


Description

This array index constant holds the index to the result of the world and view matrices multiplied together, and is one of the available pre-defined matrix uniforms that GameMaker:studio creates for you to use within the shader code editor. This is often used for things like fog. The index returned would be used as an array value when calling the gm_Matrices constant within the shader code.


Example :

vec4 pos = gm_Matrices[MATRIX_WORLD_VIEW] * objectspacepos;

The above example is a snippet of GLSL code using the MATRIX_WORLD_VIEW constant.
