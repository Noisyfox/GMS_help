Time Lines 时间线

本章节涉及时间线函数。

时间线是一个强大的机制来控制你的游戏中的事件，本节涵盖了不同的方式，以便你可以在你的游戏中处理他们。

时间线函数

如下函数是为了增加和操作时间线在你的游戏中：

  1. timeline_exists
  2. timeline_get_name
  3. timeline_add
  4. timeline_delete
  5. timeline_moment_clear

时间线实例变量

如下内置于全部实例的变量可以使时间线工作更加轻松：

  1. timeline_index
  2. timeline_running
  3. timeline_speed
  4. timeline_position
  5. timeline_loop
