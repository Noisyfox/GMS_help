DS Maps 地图

ds_map 是一个数据结构，其存储在相连对组成的一个键和一个对应的值的数据。

ds_map是一个非常有用的数据结构，因为它可以让您成对存储键和值。例如，在你的游戏中一个角色可以有不同数量的道具（键）和个别道具，他可以 有一些（值），在RPG一样，你可以有10个治疗药水，5个法力药水和100金。地图保持所有成对在一个地方，你可以匹配后添加到地图，以及使用一些简单 的函数按指定键搜索值。在使用它们之前有几件事情你应该知道。 但是！地图不能排序以任何（可识别）的方式，这意味着要找到某一个键，你可能要遍历整个事情（这是 很慢的）。也没有办法存储两个相同的键，然而你可以指定一个键有两个值。

NOTE : ds_map 函数已经在上一个GameMaker版本变更 ，所以过去的游戏可能无法正常工作或你预期的那样。

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。

存在以下函数处理DS 地图。

  1. ds_map_exists
  2. ds_map_create
  3. ds_map_add
  4. ds_map_clear
  5. ds_map_copy
  6. ds_map_replace
  7. ds_map_delete
  8. ds_map_empty
  9. ds_map_size
  10. ds_map_find_first
  11. ds_map_find_last
  12. ds_map_find_next
  13. ds_map_find_previous
  14. ds_map_find_value
  15. ds_map_read
  16. ds_map_write
  17. ds_map_destroy
