DS Priority Queues 优先队列

ds_priority 是一种数据结构，以它包含的信息的重要值为基本。

ds_priority队列是一个特殊的数据结构，类似于 ds_queue 的，只在队列的值的排序是按优先级（总是实数），是已被分配的值。这种结构可以是非常有用的，构建的排行榜或信息列表中的每个条目，按重要性进行加权

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。

存在以下函数处理ds_priority 队列：

  1. ds_priority_create
  2. ds_priority_destroy
  3. ds_priority_clear
  4. ds_priority_empty
  5. ds_priority_size
  6. ds_priority_add
  7. ds_priority_change_priority
  8. ds_priority_delete_max
  9. ds_priority_delete_min
  10. ds_priority_delete_value
  11. ds_priority_find_max
  12. ds_priority_find_min
  13. ds_priority_find_priority
  14. ds_priority_copy
  15. ds_priority_read
  16. ds_priority_write
