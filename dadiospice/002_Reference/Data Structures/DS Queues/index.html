DS Queues 队列

ds_queue 是一个先入先出的数据结构。

ds_queue有点类似 ds_stack，但它的工作原理 是在先入先出（FIFO）的基础上。第一个被推入队列的值也是第一个被推出的值。你可以认为它像一间店铺的支付队列，在那里的人，是第一个在队列中支付， 然后再离开商店。队列通常用于存储的行动或行为（在脚本的形式，例如），仍需要通过实例，但也有许多其他用途。

NOTE : 由于基于动态的资源，数据结构占用内存，不再需要的时候应该总是被销毁，以防止内存泄漏，而导致减慢或最终使您的游戏崩溃。

存在以下函数处理ds_queues：

  1. ds_queue_create
  2. ds_queue_destroy
  3. ds_queue_clear
  4. ds_queue_empty
  5. ds_queue_size
  6. ds_queue_dequeue
  7. ds_queue_enqueue
  8. ds_queue_head
  9. ds_queue_tail
  10. ds_queue_copy
  11. ds_queue_read
  12. ds_queue_write
