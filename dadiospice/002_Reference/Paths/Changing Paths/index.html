Changing Paths 改变路径

本章节涉及动态路径函数

在 GameMaker:Studio 中你不仅仅限于您使用路径编辑器 Path Editor 的路径，在游戏制作工作室。有很多的函数，可用于在运行时操作和改变这些路径，或甚至创造新的路径后然后再更改和操纵这些。

以下是用于改变路径的函数：

  1. path_add
  2. path_add_point
  3. path_change_point
  4. path_insert_point
  5. path_delete_point
  6. path_clear_points
  7. path_append
  8. path_assign
  9. path_delete
  10. path_duplicate
  11. path_flip
  12. path_mirror
  13. path_reverse
  14. path_rotate
  15. path_scale
  16. path_set_closed
  17. path_set_kind
  18. path_set_precision
  19. path_shift
