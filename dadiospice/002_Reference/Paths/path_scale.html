path_scale

Sets the scale of the path.

Syntax :

path_scale;


Returns: Real


Description

This value can be used to get or to set the scale of the current path, with a default value of 1 (no scale). Set it to 2, for example, and the path will be double in size, so if the path instructs the instance to follow a 50x50 pixel square path, it will now be a 100x100 square path.


Example :

path_scale = 1 + random(2);

The above code will make the current path scale to a random size greater between 1 times and 2 times it's original size.
