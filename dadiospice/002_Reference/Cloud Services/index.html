Cloud Services 云服务

这些函数能够被用于与不同云服务器同步

在做游戏时,常常需要存储的信息在文件中关于一些游戏状态类型,但在设备上存储这些信息并不总是最好的选择,如果玩家删除游戏,然后重新安装,这些信息可 能会 丢失。为了防止这种情况你可以使用各种不同的云服务,提供数据存储在互联网上检索和修改在任何时间。你应该注意,这个函数被限制到一个单一的数据 “blob” 每场游戏,所以每次你发送一个新的保存到云服务,无论它是一个字符串或一个文件,它将覆盖任何以前存储数据。

GameMaker:Studio 支持这几个简单的函数,结合工作 Asynchronous Cloud Event. 这些函数解释在以下页面:

  1. cloud_synchronise
  2. cloud_string_save
  3. cloud_file_save

NOTE: 目前只有亚马逊 WhisperSync 支持,您必须激活它通过 Android Tab of the Global Game Settings.
