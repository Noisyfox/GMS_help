room_get_name

Returns the name of the room with the given index.

Syntax :

room_get_name(index);


ArgumentDescription
index The index of the room to check the name of.


Returns: String


Description

This function can be used to return the name of the specified room as a string. Please note that this is only a string and cannot be used to reference the room in any way. For that you must use the room index.


Example :

if room_caption="" room_caption = room_get_name(room);

The above code will check to see if the room has a caption and if it does not it sets the caption to be the same as the name of the room.
