score 分数

全局变量用于保存游戏分数。

Syntax :

score;


Returns : Real


Description

这个变量是全局变量用于保存记录游戏分数。记住这是不一定总是如此——只是因为它叫做“分数”并不意味着它必须被用于分数…它可以容纳任何实际或整数价值你希望你的游戏。


Example :

if place_meeting(x, y, obj_Bullet)
   {
   score += 10;
   instance_destroy();
   }

上面的代码检查当前实例是否存在碰撞的实例对象索引在“obj_Bullet”,如果有,它增加了10的游戏分数,然后毁灭自己。
