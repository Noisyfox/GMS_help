Expressions 表达式

本章节涉及不同的数学表达式

表达式可以是数字（比如：4、3、5、6、等等），十六进制数字 $ 打头的格式（ 比如 $00FFAA），单引号或双引号括起来的字符串（比如"Hello"或'hello'）或者其他更复杂的表达式。如表达式中双标识符（按照优先级 ):


= - 用于赋值到变量。注意这也用于在GMS中比较变量，您将在例子和其他人的代码中看到。然而在过去版本中你也许也使用过==作为比较，= 的赋值例子请见如下：

a = 12; speed = 5; val = (old_val + 5);


&&, ||, ^^ (and, or and xor) - 结合布尔变量返回 true 或 false。如下例子如果为 true 则执行代码。

if (a == b && c == d) {do something...}// and
if (a == b || c ==d) {do something...}// or
if (a == b ^^ c == d) {do something...} // xor


<, <=, ==, !=, >, >= - 这些比较能够返回 true 或 false（true为1，false为0）。例子如下：

if (a < b) {do something...}
if (a != b) {do something...}


|, &, ^, <<, >> : 您可以执行位运算。| = 或, & = 并, ^ = 异或, << = 左移, >> = 右移. 例子如下:

x = (x & $ffffffe0) + 32;
if (y ^ $1f) > 0 {do something...};


+, -, *, / - 加减乘除，例子如下：

c = a * b;
str = a + "world";


++, -- - 加或减值。值得注意的是在值的前面和后面结果不同，例子如下：
  • ++a 加后返回结果。
  • a++ 加前返回结果再执行加。

例子如下:

var a = 1;
show_debug_message(string(a++));
show_debug_message(string(++a));

 debug 输出结果为 1 和 3. 另外例子如下:

for (i = 0; i < 10; i++) {do something...};
if hit == true score--;


div, mod (%) - 整除和取余，整除是两个整型相除去整数部分，要截尾。取余是两个整型数相除取余数，余数符号与被除数符号相同，示例如下：

secs = time mod 60;
time_str = string(time div 60);


另外还有如下一元运算符：Also, the following unary operators exist:

  • ! (not) : 否, true 转换为 false 和 false 转换为 true
  • - : 否定后面的数值
  • ~ : 否定后面的数值进行二进制位取反（1变0，0变1）

作为值您可以使用数字，变量，或函数返回值，也可以置于表达式之间。全部的操作符可以使用在实数上，但也可以用于比较和连接字符串。

Note : 与某些语言相反，两个参数之间总在运算，一个参数决定着另外一个参数。

整合例子如下:

{
x = 23;
color = $FFAA00;
str = 'hello world';
y += 5;
x *= y;
x = y << 2;
x = 23 * ((2 + 4) / sin(y));
str = 'hello' + " world";
b = (x < 5) && !(x==2 || x==4);
}
