The "Repeat" Statement

这个章节描述了如何使用 "repeat".

A "repeat" 表达形式如下 :

repeat (<expression>) <statement>


该语句声明重复整数数次的表达式。例如,以下程序创建五个球在随机位置。

{
repeat (5) instance_create(random(400), random(400), obj_ball);
}


这可以是非常有用的,以避免输入多次相同的代码,或使用数组,或用于一系列运算等等……例如:

{
var i, total;
i = 0;
total = 0;
repeat (10)
   {
   total += array[i];
   i += 1
   }
draw_text(32, 32, total);
}
