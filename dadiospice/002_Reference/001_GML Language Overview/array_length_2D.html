array_length_2D

返回一个二维数组的第二维的长度。

Syntax :

array_length_2D(array, n);


ArgumentDescription
array 用于检查的数列
n 输入数组第一维的值，以确定是要返回哪个一个一维的二维长度


Returns: Real


Description

有了这个函数，你可以得到的第二维的数组长度（条目数）。你需要提供第一维度的条目数，该函数将返回该数列的第二维度的条目的数目（找到的第一个维度的长度使用的函数 array_length_1D）。如果给定的变量不是数组是一维数组该函数将返回0。


Example :

for (var i = array_length_1D(a); i > -1; i--;)
   {
   for (var j = array_length_2D(a, i); j > -1; i--;)
      {
      a[i, j] = -1;
      }
   }

以上代码将遍历二维数组并且设定为-1.
