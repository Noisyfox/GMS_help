is_array

检查一个变量看是否是数组。

Syntax :

is_array(variable);


ArgumentDescription
variable 检查数组变量         


Returns: Boolean


Description

此函数可用于检查，看看如果一个变量是一个数组（它将返回true）或不（在这种情况下，它会返回false）。


Example :

if is_array(a)
   {
   a = 0;
   }

上面的代码检查一个变量，看它是否是一个数组，如果该函数返回true，该数组中删除。
