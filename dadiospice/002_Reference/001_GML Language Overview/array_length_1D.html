array_length_1D

返回的数组的第一维的长度。

Syntax :

array_length_1D(array);


ArgumentDescription
array 用于检查的数列


Returns: Real


Description

有了这个函数，你可以得到一个数组的长度（条目数）。将可以返回数组中的条目数量，无论数组是1D，2D，在这种情况下，第一维的长度将被返回（如果您需要找到第二维的长度，你应该使用函数 array_length_2D). 如果给定的变量不是数组该函数将返回0。


Example :

for (var i = array_length_1D(a); a > -1; i--;)
   {
   a[i] = -1;
   }

上面的代码将遍历一个数组每个条目，并设置为-1。
