argument_count

返回脚本的参数数量。

Syntax :

argument_count;


Returns : Real


Description

返回的脚本的参数数量的变量是只读变量。


Example :

var i, arg;
for (i = 0; i < 5; i += 1;)
   {
   if argument_count > i
      {
      arg[i] = argument[i]
      }
   else
      {
      arg[i] = -1;
      }
   }

在上述代码中示例了使用 argument_count 初始化变量，及如何给数组变量赋值。
