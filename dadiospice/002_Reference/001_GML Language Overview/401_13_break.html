The "Break" Statement

这个章节描述了如何使用 "break"

 "break"是用来提前结束, for, repeat, while, do... until 这些循环类型, 或者告知 switch 声明该点结束. 下面你可以看到一些例子可以使用的,其形式就是:

break;



"break" in a for loop :

{
var i;
for (i = 0; i < 10; i += 1)
   {
   if array[i] = 234 break;
   }
num = i;
}


"break" in a repeat loop :

{
var i, temp;
i = 0;
temp = 0;
repeat (10)
   {
   temp += array[i];
   if temp > max_total break else i += 1;
   }
}


"break" in a while loop :

{
var i;
i = 0;
while (!place_free(x, y))
   {
   x = random(room_width);
   y = random(room_height);
   if i > 50 break else i+=1;
   }
}
