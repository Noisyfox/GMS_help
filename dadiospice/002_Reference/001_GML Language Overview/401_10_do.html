The "Do" Statement

这个章节描述了如何使用 "do"

 "do" 完整的是 "do... until" 语句，缺一不可，表达形式如下：

do <statement> until (<expression>)


 语句(也可以是一个代码块)是执行直到表达式是发现为真,和最初的语句总是执行至少一次。小心你的做循环,你很容易地使它们永远循 环,在这种情况下你的游戏将挂起,不响应任何用户输入了。下面你可以找到一个例子,一个典型的方式使用"do... until" ：

{
do
   {
   x = random(room_width);
   y = random(room_height);
   }
until (place_free(x, y)); }


上面的程序试图将当前对象移到一个自由的位置(这样的行动也可查阅 move an object to a random position).
