The "Continue" Statement

这个章节描述了如何使用 "continue" .

"continue" 表达形式如下 :

continue


通常插入到一个循环中 (repeat, while, do... until or for), 它将立即返回循环的到下次循环开始(如果使用外循环它充当一个退出声明,退出事件)。下面是一个例子,它使用在“for”循环:

{
var i;
for (i = 0; i < 10; i += 1)
   {
   if array[i] = "" continue;
   array[i] = "";
   }
}


上面的代码如果array[i]]值已经是一个空字符串会跳过回到开始的循环。
