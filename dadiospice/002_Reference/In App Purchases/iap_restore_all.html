iap_restore_all

Restores all purchases for the game.

Syntax :

iap_restore_all();


Returns : N/A


Description

Restores all purchases to their default status for the game running the function. This would be applicable if the user has changed device or removed the title from their current device and has re-installed the title. Take care with this function as most current Apple Store/Google Play titles do not explicitly expose this ability.


Example :

iap_restore_all();

This would restore all purchase status for the game on the device running it.
