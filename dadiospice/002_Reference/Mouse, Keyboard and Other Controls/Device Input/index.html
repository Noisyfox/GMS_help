Device Input 设备输入

本章节涉及多点触摸输入的设备。

当你的游戏正在针对这些允许多点触摸平台，你应该使用这些函数。他们的工作方式与PC上的工作相同 ，如有多个鼠标连接和常量使用标准鼠标函数一样检查。

ConstantDescription
mb_left鼠标左键
mb_middle鼠标中键（不是所有的平台都有效）
mb_right鼠标右键
mb_none无鼠标键
mb_any任意鼠标键



NOTE : 在触摸屏设备上，双击返回，相当于一个鼠标右键事件。因此，如果您需要快速连续地检测多个输入，例如，你需要，检查的mb_left和mb_right。此功能可以切换或关闭使用 device_mouse_dbclick_enable.

如下您可以找到全部的设备特殊控制函数：

  1. device_mouse_check_button
  2. device_mouse_check_button_pressed
  3. device_mouse_check_button_released
  4. device_mouse_dbclick_enable
  5. device_mouse_x
  6. device_mouse_y
  7. device_mouse_raw_x
  8. device_mouse_raw_y
  9. device_is_keypad_open
  10. device_get_tilt_x
  11. device_get_tilt_y
  12. device_get_tilt_z

请注意在移动设备 (Android, iOS and Windows Phone)，能够在任何时候使用后退按钮，它对应键盘上的 backspace 在 GameMaker:Studio中 keyboard constant vk_backspace，因此您可以检测这个键，在这个返回按钮被使用时在设备上执行动作。
