device_mouse_dbclick_enable

Sets whether a double left-click should be detected as mb_right or not.

Syntax :

device_mouse_dbclick_enable(bool);


ArgumentDescription
bool Set double-click detection on (true) or off (false).


Returns: Boolean


Description

This function can be used to set the device to detect a double tap of the mb_left (left mouse button) as an mb_right (right mouse button) tap or not. By default this is set to true, meaning that everytime the user taps the device screen twice quickly and consecutively, the return value is the same as if the right mouse button had been clicked. When this is on, the first tap will be detected as mb_left, and the second as mb_right, so make sure that any code you use takes this into account.


Example :

if os_type == os_windows || os_type == os_mac
   {
   device_mouse_dbclick_enable(false);
   }

The above code checks to see if the device running the game is a Windows PC or a Mac and if it is eithor of them, it disables the double tap function.
