Joystick Input 摇杆输入

本章节涉及摇杆输入。

GameMaker:Studio 支持两个 摇杆或手柄 在游戏中使用, 一些专用函数用于设计位置及按钮按下细节。

NOTE : 这些函数仅windows支持，并且需要对过去版本的支持。像这样，它们可能不能在现在PC机上工作。

如下是关于摇杆控制的函数：

  1. joystick_exists
  2. joystick_name
  3. joystick_axes
  4. joystick_buttons
  5. joystick_has_pov
  6. joystick_direction
  7. joystick_check_button
  8. joystick_xpos
  9. joystick_ypos
  10. joystick_zpos
  11. joystick_rpos
  12. joystick_upos
  13. joystick_vpos
  14. joystick_pov
