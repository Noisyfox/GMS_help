Achievements and Leaderboards 成就和排行榜

在这里你可以找到整合您的游戏与现有的排行榜和成就的API的函数

到目标设备运行iOS或Android可以连接一个也可以到两个排行榜和成就API。适用于iOS的，你有iTunes的游戏中心，和 Android的，你可以使用亚马逊的游戏圈，都使用此页上找到的函数。这些函数允许你的游戏发送分数和成就信息到你已经为游戏设置了的开发者的帐户的排 行榜，是一个很好的方式让玩家参与游戏的社会性的一面，因为他们可以看到在排行榜上其他人在你的游戏中已经实现了的分数和成就的信息，并尝试超越他们。注 意：这些功能与iOS，你必须有一个有效的游戏中心帐户，并在iOS 应用程序ID为您的游戏。注意：对于这些功能的工作的Android则必须到亚马逊的应用程序商店提交您的游戏，并建立了必要的区段以及API激活你的游 戏在 全球游戏设置。 下面列出的是将您的游戏，在游戏中心的功能：

NOTE : 为了让这些函数工作在ios，你必须有一个有效的额游戏中心账户和你游戏的ios app id。
NOTE : 为了让这些函数工作在 Android，你必须提交你的游戏到亚马逊应用商店，并且设置必须的部分如为你的游戏使用API而在 Global Game Settings 里激活。

下面列出的是将您的游戏与游戏中心整合的函数：


  1. achievement_login_status
  2. achievement_login
  3. achievement_logout
  4. achievement_available
  5. achievement_post
  6. achievement_post_score
  7. achievement_show_achievements
  8. achievement_show_leaderboards
