physics_fixture_set_friction

This function sets the friction of a fixture.

Syntax :

physics_fixture_set_friction(fixture, friction)


ArgumentDescription
fixture the index of the fixture
friction the friction of the fixture must be between 0 and 1


Returns : N/A


Description

Friction is the force that resists the relative motion of material elements sliding against each other, which in the GameMaker:Studio physics world, translates as the loss of momentum caused by the collision of two instances with fixtures bound to them. So, when two instances collide, their motion is affected by this value, with a high friction causing a larger loss of momentum than a lower value.


Example :

physics_fixture_set_friction(fix_Brick, 0.1);

The code above will set the friction of the fixture indexed in "fix_brick" to 0.1.
