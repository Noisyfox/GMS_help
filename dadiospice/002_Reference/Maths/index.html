Maths 数学

本章节列出了数学函数

GameMaker:Studio 包含了大量的函数来处理不同领域的数学。大多数这些函数可以在下面列出的章节中找到 :

  1. Real Number Functions 实数函数
  2. Vector Functions 矢量函数

GameMaker:Studio 也接受布尔 boolean 变量为 true (1) 和 false (0) 和 圆周率 pi (3.141592653589793280(etc)...).
