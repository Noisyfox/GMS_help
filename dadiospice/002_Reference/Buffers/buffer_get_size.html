buffer_get_size

Get the size (in bytes) of the given buffer.

Syntax :

buffer_get_size(index);


ArgumentDescription
index The index of the buffer to get the size of.


Returns : Real


Description

With this function you can get the size of the given buffer in bytes.


Example :

var temp_buff = buffer_create(buffer_get_size(buff), buffer_fixed, 0);

The above code will create a new buffer and store it's index in the local variable "temp_buff", with size of this new buffer being the same as that of the buffer indexed in the variable "buff".
