buffer_get_surface

Copy one buffer (or part of) to another.

Syntax :

buffer_get_surface(index, surface, mode, offset, modulo);


ArgumentDescription
index The index of the buffer to use.
surface The index of the surface to use.
mode The data to get from the surface (see the list of constants below).
offset The data offset value.
modulo


Returns : N/A


Description

With this function you can write the information from a surface to a given buffer. Both the surface and the buffer must have been created previously. You can choose the copy "mode" for the data being stored which is set by the constants listed below, as well as the offset and modulo values.

ConstantDescription
buffer_surface_copyCopies all data from the surface.
buffer_surface_grayscaleOnly copies the saturation values from the surface.
buffer_surface_mask


Example :

buffer_get_surface(buff, screen_surf, buffer_surface_copy, 0, 0);

The above code will copy the data stored in the surface indexed in the variable "screen_surf" to the buffer indexed in the variable "buff" with no offset.
